CRISPR dependency map analysis summary for ds001_depmap

I analyzed the 50,000-row local dataset using adjusted linear models, interaction screens, and held-out subgroup validation. Dependency scores are on the dataset's natural scale, where more negative values indicate stronger dependency. Continuous predictors were standardized for adjusted regression coefficients. The main adjusted model included molecular binary features, standardized continuous features, lineage, lineage_subtype, culture_type, and screen_batch. For heterogeneity, I screened driver-by-modifier interactions after residualizing against all main effects and applied BH FDR correction within each screen. I also ran a systematic subgroup search for every dependency outcome using a deterministic even/odd discovery-validation split; discovery screened single predicates, pairs, and selected triples, while the reported subgroup tests below use the held-out validation split.

Canonical oncogene dependencies were strongly supported. `braf_v600e=1` predicted more negative `dependency_BRAF` with adjusted effect -0.5543 (p underflowed to 0). `egfr_amplification=1` predicted more negative `dependency_EGFR` with adjusted effect -0.3492 (p=0). `erbb2_amplification=1` predicted more negative `dependency_ERBB2` with adjusted effect -0.4327 (p=0). `kras_mutation=1` predicted more negative `dependency_KRAS` with adjusted effect -0.3097 (p=0). `nf1_loss=1` predicted more negative `dependency_RIT1` with adjusted effect -0.0954 (p=6.87e-69). Subsequent driver-by-modifier screens did not find robust FDR-significant modifiers for these five canonical alteration dependencies, so their best-supported subgroup definitions remained the alteration-positive groups themselves.

Growth-rate effects were broad but varied in magnitude. A one-SD increase in `growth_rate_doublings_per_day` had the strongest adjusted association with `dependency_POLR2A` (effect -0.0614, p=0), and a held-out high-growth subgroup using `growth_rate_doublings_per_day>=0.965` had mean `dependency_POLR2A` -0.4963 versus -0.3893 in other validation lines (effect -0.1069, p=8.33e-94). Growth also had smaller more-negative associations with `dependency_TMED10` (adjusted effect -0.0203, p=2.25e-38) and `dependency_DHX9` (adjusted effect -0.0215, p=1.08e-42). For TMED10, the best multi-feature rule from subgroup search (`cas9_activity_score>=1.08`, high growth, and `mutation_burden<=3.25`) was not significant in validation (effect -0.0310, p=0.092), so I treated TMED10 as a weak growth-associated effect rather than a validated molecular subgroup.

The strongest heterogeneity result involved `dependency_KIF18A`. Main effects implicated `lineage=colorectal` (adjusted effect -0.3714, p=0), `apc_mutation=1` (effect -0.1221, p=7.51e-115), and high `wnt_activity_score` (per-SD effect -0.0288, p=1.82e-35). `smad4_loss=1` moved scores in the opposite direction (adjusted effect +0.0938, p=1.30e-69). Interaction screening confirmed that APC-associated KIF18A dependency was amplified in colorectal lines (`apc_mutation * lineage=colorectal` effect -0.6814, p=0) and with WNT activity (`apc_mutation * wnt_activity_score_z` effect -0.2159, p=0). SMAD4 loss suppressed this dependency: within APC-mutant colorectal high-WNT validation lines, `smad4_loss=1` had mean KIF18A dependency -0.2372 versus -1.3128 for `smad4_loss=0` (difference +1.0756, p=1.30e-186). The final complete KIF18A subgroup was `apc_mutation=1`, `lineage=colorectal`, `wnt_activity_score>=0.821`, and `smad4_loss=0`, with validation effect -1.0574 versus all other lines (p=0).

MYC-linked dependencies were supported for RPL11 and DHX9. For `dependency_RPL11`, `copy_number_8q24` had an adjusted per-SD effect of -0.0558 (p=1.35e-120). A refined validation subgroup, `myc_amplification=1`, `growth_rate_doublings_per_day>=0.965`, and `ploidy>=3.068`, had mean `dependency_RPL11` -0.4219 versus -0.2680 in other validation lines (effect -0.1539, p=5.41e-12), and within MYC-amplified lines this high-growth/high-ploidy state was still more negative by -0.0663 (p=0.00289). For `dependency_DHX9`, `rnaseq_MYC_log2_tpm` had an adjusted per-SD effect of -0.0170 (p=1.86e-24). The final DHX9 subgroup was high growth, high MYC expression, and low AXL expression (`growth_rate_doublings_per_day>=0.965`, `rnaseq_MYC_log2_tpm>=5.674`, `rnaseq_AXL_log2_tpm<=2.372`), with validation effect -0.0627 (p=0.000462).

Final best-supported dependency subgroup conclusions: BRAF dependency is defined by `braf_v600e=1`; EGFR by `egfr_amplification=1`; ERBB2 by `erbb2_amplification=1`; KRAS by `kras_mutation=1`; RIT1 by `nf1_loss=1`; KIF18A by `apc_mutation=1`, `lineage=colorectal`, high WNT, and absence of SMAD4 loss; TMED10 shows only a small high-growth association with no validated multi-feature subgroup; DHX9 is strongest in high-growth, high-MYC, low-AXL lines; POLR2A is primarily a high-growth dependency; and RPL11 is strongest in MYC-amplified, high-growth, high-ploidy lines.
