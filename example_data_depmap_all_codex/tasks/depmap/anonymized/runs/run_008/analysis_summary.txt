CRISPR dependency map analysis summary for ds001_depmap

Data and analysis approach
I analyzed 50,000 cell-line records with 36 anonymized cell-line features and 10 dependency outcomes. More negative dependency scores indicate stronger knockout dependency. I decoded the local Parquet bundle only, then ran main-effect screens across every feature-outcome pair and a systematic dependency-heterogeneity search for each outcome. The heterogeneity search evaluated single feature predicates and the strongest two-feature predicate combinations; p-values are large-sample Welch or OLS tests and many underflow below 1e-300 because of the sample size.

Main effects supported in early iterations
BRAF, EGFR, ERBB2, KRAS, and RIT1 each had clear binary-feature associations. eature_009 == 1 was associated with more negative dependency_BRAF (effect -0.555, mean -0.805 vs -0.250). eature_022 == 1 was associated with more negative dependency_EGFR (effect -0.349). eature_024 == 1 was associated with more negative dependency_ERBB2 (effect -0.432). eature_014 == 1 was associated with more negative dependency_KRAS (effect -0.307). eature_027 == 1 was associated with more negative dependency_RIT1 (effect -0.096). All were statistically significant.

Shared screen-covariate patterns
eature_016 behaved like a broad dependency modifier: higher values were associated with more negative dependency_POLR2A (slope -0.305), dependency_DHX9 (slope -0.109), and dependency_TMED10 (slope -0.101). dependency_RPL11 was more negative with higher eature_003 (slope -0.102) and with eature_018 == 1 (effect -0.109).

KIF18A subgroup discovery
The strongest and most biologically distinctive pattern was for dependency_KIF18A. Main effects showed that eature_011 == colorectal had much more negative KIF18A dependency than other categories (effect -0.468, mean -0.720 vs -0.252). eature_035 == 1 was also associated with stronger KIF18A dependency (effect -0.292), and higher eature_028 had a negative slope (slope -0.106). The refined subgroup screen found the complete best-supported subgroup: eature_011 == colorectal AND feature_035 == 1, with mean dependency_KIF18A -0.943 versus -0.253 in all other lines, effect -0.690, n=3925, p<1e-300. A related subgroup, eature_011 == colorectal AND feature_028 >= 0.819, was nearly as strong (effect -0.676).

Systematic heterogeneity results by outcome
For dependency_BRAF, the best negative two-feature subgroup was eature_009 == 1 AND feature_010 == not_applicable (effect -0.540). This largely preserved the eature_009 main effect rather than revealing an independent interaction. For dependency_EGFR, the best subgroup was eature_022 == 1 AND feature_014 == 0 (effect -0.346). For dependency_ERBB2, it was eature_024 == 1 AND feature_030 == 0 (effect -0.427). For dependency_KRAS, it was eature_014 == 1 AND feature_020 == 0 (effect -0.292). For dependency_RIT1, it was eature_027 == 1 AND feature_021 == 0 (effect -0.096).

For the broader essentiality-like outcomes, subgroup effects were smaller but significant. dependency_DHX9 was more negative in eature_016 >= 0.964 AND feature_023 >= 5.669 (effect -0.059). dependency_TMED10 was more negative in eature_016 >= 0.964 AND feature_032 == 0 (effect -0.035). dependency_POLR2A was more negative in eature_016 >= 0.964 AND feature_012 == 0 (effect -0.102). dependency_RPL11 was more negative in eature_003 >= 0.428 AND feature_018 == 1 (effect -0.112).

Overall conclusion
The dataset contains several strong single-feature dependency markers plus one particularly strong heterogeneous dependency pattern. The clearest final hypothesis is that dependency_KIF18A is concentrated in the subgroup defined by eature_011 == colorectal and eature_035 == 1; this subgroup has a much stronger negative dependency score than the rest of the screen. Other dependency outcomes are mostly explained by one dominant feature plus weaker modifiers, with statistically significant but smaller two-feature refinements.
