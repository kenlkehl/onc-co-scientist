CRISPR Dependency Map analysis summary for ds001_depmap

The task bundle contains 50000 cell-line records, 36 anonymized cell-line features, and 10 dependency outcomes. More negative dependency scores were treated as stronger dependency after knockout. The analysis used signed subgroup contrasts for binary and categorical predicates, threshold contrasts for continuous predicates, and systematic dependency-heterogeneity searches over pair and triple subgroup definitions for every dependency outcome.

Iteration 1 tested BRAF. The supported main hypothesis was that `feature_009 == 1` marks stronger `dependency_BRAF`: mean -0.8047 versus -0.2495 outside the subgroup, effect -0.5552, p <= 1e-300. A refined pair with `feature_030 == 0` was also significant, but its effect (-0.5514) did not improve meaningfully on `feature_009 == 1`; the best BRAF rule is therefore the simpler `feature_009 == 1` marker.

Iteration 2 tested EGFR. `feature_022 == 1` strongly marked EGFR dependency, with mean -0.6010 versus -0.2524 outside, effect -0.3486, p <= 1e-300. The heterogeneity search supported a more concentrated subgroup, `feature_022 == 1`, `feature_023 >= 5.669`, and `feature_015 <= 3.230`, with mean -0.6714 versus -0.2775, effect -0.3939. This refined hypothesis was supported, although the signal remained anchored by `feature_022`.

Iteration 3 tested ERBB2. `feature_024 == 1` was the dominant dependency marker: mean `dependency_ERBB2` -0.6856 versus -0.2535, effect -0.4321, p <= 1e-300. The supported refined subgroup was `feature_024 == 1`, `feature_011 == breast`, and `feature_013 >= 0.519`, which had mean -0.7597 versus -0.2930 outside and effect -0.4667.

Iteration 4 tested KRAS. The main supported hypothesis was `feature_014 == 1`, with mean -0.5594 versus -0.2520, effect -0.3075, p <= 1e-300. Pancreatic lineage (`feature_011 == pancreatic`) was also significant but smaller, effect -0.1127. The heterogeneity search supported `feature_014 == 1`, `feature_016 >= 1.088`, and `feature_028 <= -0.406` as the strongest KRAS subgroup, with mean -0.6640 versus -0.3230 and effect -0.3410.

Iteration 5 tested RIT1. `feature_027 == 1` was supported as a RIT1 dependency marker, mean -0.3527 versus -0.2572, effect -0.0955, p <= 1e-300. The refined subgroup `feature_027 == 1`, `feature_011 == brain`, and `feature_008 == 0` was also supported and more concentrated, with mean -0.4315 versus -0.2651, effect -0.1665, p=3.33e-15.

Iteration 6 tested KIF18A and found the clearest multivariable dependency pattern. Colorectal lineage (`feature_011 == colorectal`) was strongly associated with more negative `dependency_KIF18A` (effect -0.4680), and high `feature_028 >= 1.427` was also associated (effect -0.3773). The complete supported subgroup was `feature_011 == colorectal`, `feature_028 >= 1.427`, `feature_020 == 0`, and `feature_035 == 1`; this group had mean -1.3952 versus -0.2678 outside, effect -1.1274, p <= 1e-300. Suppressor checks refuted the idea that colorectal/high-`feature_028` alone is sufficient: within that background, `feature_020 == 1` and `feature_035 == 0` each had near-background KIF18A dependency.

Iteration 7 tested TMED10. The supported main hypothesis was high `feature_016 >= 1.088`, with mean -0.2906 versus -0.2499, effect -0.0407, p=1.11e-14. The heterogeneity search supported a stronger subgroup, `feature_016 >= 1.088`, `feature_026 >= 1.153`, and `feature_004 == adherent`, with mean -0.3578 versus -0.2532 and effect -0.1046.

Iteration 8 tested DHX9. High `feature_016 >= 1.088` was supported, effect -0.0433, p=4.44e-16, and high `feature_023 >= 6.330` was also supported, effect -0.0314, p=1.28e-9. The refined supported subgroup was `feature_016 >= 1.088`, `feature_023 >= 5.669`, and `feature_020 == 1`, with mean -0.4472 versus -0.3332 and effect -0.1139.

Iteration 9 tested POLR2A. High `feature_016 >= 1.088` was the dominant supported marker, with mean -0.5281 versus -0.4070, effect -0.1211, p <= 1e-300. A refined subgroup, `feature_016 >= 1.088`, `feature_018 == 1`, and `feature_005 == batch_c`, was also supported, with mean -0.5982 versus -0.4181 and effect -0.1801.

Iteration 10 tested RPL11. High `feature_003 >= 1.003` was supported as the strongest RPL11 marker, with mean -0.3831 versus -0.2573, effect -0.1257, p <= 1e-300. `feature_018 == 1` was also supported, effect -0.1091. The strongest refined subgroup combined `feature_003 >= 1.003`, `feature_016 >= 0.964`, and `feature_033 >= 0.602`, giving mean -0.4600 versus -0.2684 and effect -0.1916.

Overall, BRAF, EGFR, ERBB2, KRAS, and RIT1 were each primarily driven by a dominant binary marker (`feature_009`, `feature_022`, `feature_024`, `feature_014`, and `feature_027`, respectively), with refined subgroups mostly concentrating those already strong effects. TMED10, DHX9, and POLR2A showed smaller but robust high-`feature_016` dependency gradients, while RPL11 was most tied to high `feature_003`. The most biologically notable heterogeneity was KIF18A: the dependency required a complete colorectal/high-`feature_028`/`feature_020 == 0`/`feature_035 == 1` subgroup, and either `feature_020 == 1` or `feature_035 == 0` suppressed the effect back toward background.
