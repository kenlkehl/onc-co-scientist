CRISPR dependency map analysis for ds001_depmap

I analyzed 50,000 cell-line records with 36 anonymized features and ten dependency outcomes. More negative dependency scores indicate stronger knockout dependency. All tests used only the local dataset bundle. I used Welch two-sample tests for binary or subgroup contrasts, linear regression for continuous main effects, multivariable OLS for adjusted marker checks, and a systematic predicate beam search over one-, two-, and three-feature subgroup definitions for every dependency outcome.

The strongest main-effect findings were marker-like. `feature_009 == 1` was strongly associated with BRAF dependency: mean `dependency_BRAF` was -0.804698 versus -0.249503 in other lines, effect -0.555195, p underflowed to 0. `feature_022 == 1` similarly marked EGFR dependency (effect -0.348638, p=0), `feature_024 == 1` marked ERBB2 dependency (effect -0.432122, p=0), `feature_014 == 1` marked KRAS dependency (effect -0.307456, p=0), and `feature_027 == 1` marked RIT1 dependency (effect -0.095520, p=1.55e-68). These effects remained negative in multivariable OLS adjusted for lineage, subtype, culture, batch, `feature_001`, and all other numeric features.

Lineage was informative but usually secondary to the marker features. Skin lineage (`feature_011 == skin`) was more BRAF-dependent (effect -0.178251, p=2.55e-151), brain lineage was more EGFR-dependent (effect -0.055171, p=1.14e-16), breast lineage was more ERBB2-dependent (effect -0.029180, p=6.56e-09), and pancreatic lineage was more KRAS-dependent (effect -0.112747, p=1.33e-86). For BRAF, interaction testing showed that skin did not materially modify `feature_009`; `feature_009 == 1` was sufficient.

KIF18A showed the clearest multi-feature dependency. Colorectal lineage alone had a large negative effect on `dependency_KIF18A` (mean -0.720354 vs -0.252397, effect -0.467957, p=0). `feature_035 == 1` was also negative (effect -0.292118, p=0), and higher `feature_028` was associated with more negative KIF18A scores (slope -0.097271 per SD, p=0). The systematic subgroup search identified a much sharper subgroup: `feature_011 == colorectal`, `feature_020 == 0`, and `feature_028 >= 1.427` had mean `dependency_KIF18A` -1.240923 versus -0.267666 in all other lines, effect -0.973258, p=0. Within colorectal high-`feature_028` lines, `feature_020 == 1` almost abolished this dependency (mean -0.264228 versus -1.240923 for `feature_020 == 0`, positive suppressor effect +0.976696, p=5.73e-285). Interaction OLS supported this structure, with a strong negative colorectal:high-`feature_028` term and positive terms involving `feature_020`.

The systematic dependency-heterogeneity search was run for every outcome. Best-supported complete subgroup summaries were: BRAF, `feature_009 == 1` (effect -0.555195); EGFR, `feature_022 == 1`, `feature_016 >= 0.964`, and `feature_003 >= 0.428` (effect -0.391416); ERBB2, `feature_024 == 1`, breast lineage, and `feature_013 >= 0.519` (effect -0.466652); KRAS, `feature_014 == 1`, `feature_016 >= 1.088`, and `feature_028 <= -0.406` (effect -0.341004); RIT1, `feature_027 == 1` and brain lineage (effect -0.139699); KIF18A, colorectal, `feature_020 == 0`, and `feature_028 >= 1.427` (effect -0.973258); TMED10, adherent culture, `feature_016 >= 1.088`, and `feature_026 >= 1.153` (effect -0.104617); DHX9, `feature_015 <= 2.083`, `feature_016 >= 0.964`, and `feature_023 >= 5.669` (effect -0.104264); POLR2A, batch C, `feature_016 >= 1.088`, and `feature_018 == 1` (effect -0.180101); RPL11, `feature_003 >= 1.003`, `feature_016 >= 0.964`, and `feature_033 >= 0.602` (effect -0.191626).

For the non-oncogene outcomes, the evidence was more covariate-linked and partly additive. Higher `feature_016` was associated with more negative TMED10, DHX9, and POLR2A scores. POLR2A was especially tied to high `feature_016`: `feature_016 >= 1.088` alone had effect -0.121093 (p=2.94e-114), while extra predicates from the search did not have significant interaction terms. RPL11 had a broad high-`feature_003` signal (effect -0.125749, p=5.46e-120), with high `feature_016` and high `feature_033` defining a more extreme tail. TMED10 had weaker but significant joint high-`feature_016`/high-`feature_026` evidence, including a negative interaction coefficient. DHX9 was supported mainly by additive high-`feature_016` and high-`feature_023` effects, with low `feature_015` marking a smaller, more negative tail.

Overall, the most robust conclusions are: five dependency outcomes are dominated by single binary marker features (`feature_009` for BRAF, `feature_022` for EGFR, `feature_024` for ERBB2, `feature_014` for KRAS, and `feature_027` for RIT1); KIF18A has a strong, biologically meaningful-looking three-feature subgroup with an explicit suppressor (`feature_020 == 1`); and TMED10, DHX9, POLR2A, and RPL11 show smaller dependency shifts linked to continuous covariates, especially `feature_016` and `feature_003`.
