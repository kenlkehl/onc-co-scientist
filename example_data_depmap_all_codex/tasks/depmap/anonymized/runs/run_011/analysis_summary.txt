CRISPR Dependency Map analysis summary for ds001_depmap

I analyzed 50,000 cell-line records using only the local task bundle. More negative dependency scores were treated as stronger dependency. The workflow used Welch-style subgroup contrasts for binary/categorical predicates, Pearson/linear-regression scans for continuous features, and a systematic heterogeneity search over 120 candidate predicates per dependency outcome, including pair-subgroup screens and refined triple-subgroup checks. Thresholds such as 1.088 or 1.427 are empirical decile or quartile cut points from this dataset.

BRAF: The clearest BRAF dependency marker was feature_009 == 1. These lines had mean dependency_BRAF -0.8047 versus -0.2495 in feature_009 == 0 lines, effect -0.5552, p <= 1e-300. Pair searches retained feature_009 as the driver; feature_009 == 1 and feature_030 == 0 had nearly the same effect (-0.5514), so the best-supported BRAF rule is feature_009 == 1 rather than a more complex subgroup.

EGFR: feature_022 == 1 strongly marked EGFR dependency, with mean dependency_EGFR -0.6010 versus -0.2524 outside the subgroup, effect -0.3486, p <= 1e-300. A refined subgroup, feature_022 == 1 plus feature_023 >= 5.669 and feature_015 <= 3.230, was more concentrated (mean -0.6714; effect -0.3939), supporting heterogeneity around the feature_022 signal.

ERBB2: feature_024 == 1 was the dominant ERBB2 marker, mean -0.6856 versus -0.2535, effect -0.4321, p <= 1e-300. The strongest refined subgroup was feature_024 == 1, feature_011 == breast, and feature_013 >= 0.519, mean -0.7597 versus -0.2930, effect -0.4667. This supports an ERBB2 dependency concentrated within a breast/high-feature_013 subset of feature_024-positive lines.

KRAS: feature_014 == 1 was the strongest KRAS marker, mean -0.5594 versus -0.2520, effect -0.3075, p <= 1e-300. Pancreatic lineage (feature_011 == pancreatic) was also associated with more negative KRAS dependency (effect -0.1127). The refined subgroup feature_014 == 1, feature_016 >= 1.088, and feature_028 <= -0.406 had mean -0.6640, effect -0.3410, indicating stronger dependency in a high-feature_016/low-feature_028 subset.

RIT1: feature_027 == 1 was the main RIT1 marker, mean -0.3527 versus -0.2572, effect -0.0955, p <= 1e-300. Heterogeneity search found a more concentrated subgroup: feature_027 == 1, feature_011 == brain, and feature_008 == 0 had mean -0.4315 versus -0.2651, effect -0.1665, p = 3.33e-15.

KIF18A: This was the strongest multivariable dependency pattern. Colorectal lines (feature_011 == colorectal) had much more negative dependency_KIF18A than other lines (effect -0.4680), and high feature_028 (>= 1.427) was also strongly associated (effect -0.3773). The complete rule was feature_011 == colorectal, feature_028 >= 1.427, feature_020 == 0, and feature_035 == 1. This subgroup had mean dependency_KIF18A -1.3952 versus -0.2678 outside the subgroup, effect -1.1274, p <= 1e-300. Suppressor checks supported the full rule: within colorectal high-feature_028 lines, feature_020 == 1 had near-background mean -0.2642, and feature_035 == 0 had near-background mean -0.2849.

TMED10: TMED10 showed a weaker but significant broad pattern around feature_016. Lines with feature_016 >= 1.088 had mean -0.2906 versus -0.2499, effect -0.0407, p = 1.11e-14. The strongest refined subgroup was feature_016 >= 1.088, feature_026 >= 1.153, and feature_004 == adherent, mean -0.3578 versus -0.2532, effect -0.1046, p = 1.03e-9.

DHX9: High feature_016 and high feature_023 both marked DHX9 dependency. feature_016 >= 1.088 had effect -0.0433, and feature_023 >= 6.330 had effect -0.0314. The refined subgroup feature_016 >= 1.088, feature_023 >= 5.669, and feature_020 == 1 had mean -0.4472 versus -0.3332, effect -0.1139, p = 6.98e-6.

POLR2A: The POLR2A signal was dominated by feature_016. Lines with feature_016 >= 1.088 had mean -0.5281 versus -0.4070, effect -0.1211, p <= 1e-300. A refined subgroup, feature_016 >= 1.088, feature_018 == 1, and feature_005 == batch_c, had mean -0.5982 versus -0.4181, effect -0.1801, suggesting additional heterogeneity within high-feature_016 lines.

RPL11: High feature_003 was the strongest RPL11 marker, mean -0.3831 versus -0.2573, effect -0.1257, p <= 1e-300. feature_018 == 1 also supported RPL11 dependency, effect -0.1091. The most concentrated refined subgroup combined feature_003 >= 1.003, feature_016 >= 0.964, and feature_033 >= 0.602, mean -0.4600 versus -0.2684, effect -0.1916, p <= 1e-300.

Overall conclusion: Several dependencies were driven by single strong binary markers: feature_009 for BRAF, feature_022 for EGFR, feature_024 for ERBB2, feature_014 for KRAS, and feature_027 for RIT1. Other outcomes showed broader continuous-feature gradients, especially feature_016 for TMED10, DHX9, and POLR2A, and feature_003 for RPL11. The most biologically notable heterogeneity pattern was KIF18A, where the dependency was not simply colorectal or high feature_028 alone but required the complete subgroup feature_011 == colorectal, feature_028 >= 1.427, feature_020 == 0, and feature_035 == 1; either feature_020 == 1 or feature_035 == 0 suppressed the dependency back toward background.
