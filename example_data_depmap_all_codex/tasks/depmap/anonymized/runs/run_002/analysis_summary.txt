Dataset ds001_depmap was analyzed from the local dataset.parquet file only. I decoded the Arrow/Parquet data in-place, then ran univariate feature screens, lineage/categorical subgroup comparisons, continuous-feature quartile comparisons, and a systematic pairwise heterogeneity search across every dependency outcome. Effect estimates below are signed differences in mean dependency score on the natural outcome scale; negative values mean stronger dependency. P-values are exploratory Welch/normal-approximation screening p-values.

The first pass recovered several strong single-feature dependency patterns. `dependency_BRAF` was much more negative in `feature_009 == 1` cell lines than in `feature_009 == 0` cell lines (mean -0.804698 versus -0.249503; effect -0.555195; p < 1e-300). `dependency_EGFR` was more negative for `feature_022 == 1` (effect -0.348638; p < 1e-300). `dependency_ERBB2` was more negative for `feature_024 == 1` (effect -0.432122; p < 1e-300). `dependency_KRAS` was more negative for `feature_014 == 1` (effect -0.307456; p < 1e-300).

Lineage and categorical refinements showed additional smaller patterns. `feature_011 == "skin"` strengthened `dependency_BRAF` (effect -0.178251; p < 1e-300). `feature_011 == "pancreatic"` strengthened `dependency_KRAS` (effect -0.112747; p < 1e-300). `feature_011 == "brain"` modestly strengthened `dependency_EGFR` (effect -0.055171; p = 1.11e-16). These effects were supported, but they were smaller than the main binary driver effects.

Other dependency outcomes had mixed main-effect structure. `dependency_RIT1` was more negative for `feature_027 == 1` (effect -0.095520; p < 1e-300). `dependency_POLR2A` was strongly associated with high `feature_016`: top-quartile `feature_016 >= 0.964` had effect -0.104346 (p < 1e-300), and the linear slope was also strongly negative. `dependency_RPL11` was more negative for `feature_018 == 1` (effect -0.109056; p < 1e-300) and for top-quartile `feature_003 >= 0.428` (effect -0.098753; p < 1e-300). `dependency_TMED10` and `dependency_DHX9` showed weaker but significant continuous-feature patterns, especially high `feature_016` for TMED10 and high `feature_023` for DHX9.

The largest heterogeneous signal was for `dependency_KIF18A`. Main screens first identified `feature_011 == "colorectal"` (effect -0.467957), `feature_035 == 1` (effect -0.292118), and high `feature_028 >= 0.819` (effect -0.216810), all with p < 1e-300. Pairwise tests showed much stronger concentration: colorectal plus `feature_035 == 1` had effect -0.689855, and colorectal plus high `feature_028` had effect -0.676145.

Further refinement identified the complete KIF18A subgroup: `feature_011 == "colorectal"`, `feature_035 == 1`, `feature_028 >= 0.819`, and `feature_020 == 0`. This subgroup had mean `dependency_KIF18A` -1.299007 versus -0.253046 outside the subgroup (effect -1.045960; p < 1e-300). The same colorectal/`feature_035`/high-`feature_028` context with `feature_020 == 1` had mean -0.255748, indicating that `feature_020 == 1` suppresses or disables the KIF18A dependency. Top-quartile `feature_017 >= 3.559` further deepened the complete subgroup slightly (mean -1.332042; effect -1.057160 versus all others), but it was not required for the main subgroup definition.

The systematic dependency-heterogeneity search was run for each outcome. The strongest negative pairwise subgroups were: BRAF, `feature_009 == 1` and `feature_030 == 0` (effect -0.551441); EGFR, `feature_016 >= 0.964` and `feature_022 == 1` (effect -0.352601); ERBB2, `feature_022 == 0` and `feature_024 == 1` (effect -0.428575); KRAS, `feature_014 == 1` and `feature_021 == 0` (effect -0.301437); RIT1, `feature_011 == "brain"` and `feature_027 == 1` (effect -0.139699); KIF18A, the refined colorectal/`feature_035`/high-`feature_028`/`feature_020 == 0` subgroup (effect -1.045960); TMED10, `feature_011 == "prostate"` and `feature_016 >= 0.964` (effect -0.053308); DHX9, `feature_010 == "squamous"` and `feature_023 >= 5.669` (effect -0.065589); POLR2A, `feature_016 >= 0.964` and `feature_027 == 0` (effect -0.105274); RPL11, `feature_016 >= 0.964` and `feature_018 == 1` (effect -0.130333).

Several explored hypotheses were not supported. `feature_010 == "her2_enriched"` did not significantly alter `dependency_EGFR` despite a positive effect estimate (effect 0.018127; p = 0.111). Continuous `feature_019` was not significantly associated with less negative `dependency_RIT1` (effect 0.015717; p = 0.104). `feature_032 == 1` did not significantly change `dependency_TMED10` (effect 0.009024; p = 0.107). `feature_010 == "basal"` did not significantly alter `dependency_POLR2A` (effect 0.012270; p = 0.120).

Overall, most dependency outcomes were dominated by one clear binary or continuous feature with modest lineage or pairwise heterogeneity. KIF18A was the exception: its dependency was highly concentrated in a specific colorectal molecular subgroup and was nearly absent when `feature_020 == 1` within that same subgroup context.
