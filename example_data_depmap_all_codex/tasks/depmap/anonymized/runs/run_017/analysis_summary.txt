CRISPR dependency map analysis summary for ds001_depmap

I treated more negative dependency scores as stronger knockout dependency. Effect estimates below are subgroup mean minus complement mean, so negative effects indicate stronger dependency in the named subgroup. P-values are large-sample Welch-style z-test approximations; values reported as p<1e-299 are capped at 1e-300 in transcript.json.

The strongest single-marker dependencies were clear. `feature_009 == 1` marked stronger `dependency_BRAF` (effect -0.555), `feature_022 == 1` marked stronger `dependency_EGFR` (effect -0.349), `feature_024 == 1` marked stronger `dependency_ERBB2` (effect -0.432), `feature_014 == 1` marked stronger `dependency_KRAS` (effect -0.307), and `feature_027 == 1` marked stronger `dependency_RIT1` (effect -0.096). All were highly significant.

Several outcomes showed broader dependency programs. High `feature_016` (>=0.964) was associated with more negative `dependency_POLR2A` (effect -0.104) and `dependency_TMED10` (effect -0.037). For `dependency_DHX9`, high `feature_016` plus high `feature_023` (>=5.669) gave a stronger negative subgroup effect (-0.059). For `dependency_RPL11`, the combination `feature_003 >= 0.428` and `feature_018 == 1` was the clearest concentrated subgroup (effect -0.112).

The main iterative discovery was `dependency_KIF18A`. Initial main effects implicated colorectal lineage (`feature_011 == "colorectal"`, effect -0.468), `feature_035 == 1` (effect -0.292), and high `feature_028` (>=0.819, effect -0.217). Combining colorectal lineage with `feature_035 == 1` strengthened the effect (-0.690), and adding high `feature_028` strengthened it further (-0.859).

A refined threshold search inside the colorectal/`feature_035 == 1` subgroup showed that the complete KIF18A dependency requires `feature_020 == 0` and `feature_028 >= 1.0`. That four-predicate subgroup had n=2357, mean `dependency_KIF18A` -1.394 versus -0.253 in the complement, effect -1.141, p<1e-299. The signal was absent or reversed when `feature_020 == 1` despite colorectal/`feature_035 == 1`/`feature_028 >= 1.0` (effect 0.054) or when `feature_028 < 1.0` despite colorectal/`feature_035 == 1`/`feature_020 == 0` (effect 0.034).

Overall, the dataset contains strong single-feature oncogene dependency signatures, broader essential-gene dependency programs tied to continuous features, and one standout buried multi-feature interaction: `dependency_KIF18A` is concentrated in cell lines with `feature_011 == "colorectal"`, `feature_035 == 1`, `feature_020 == 0`, and `feature_028 >= 1.0`.
