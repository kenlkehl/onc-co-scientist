CRISPR dependency analysis summary for ds001_depmap

I analyzed the 50,000-row DepMap-style screen using only the local parquet file and local column description. More negative dependency scores indicate stronger dependency after knockout. I decoded the parquet data, screened all feature-outcome main effects, and then ran systematic one-, two-, and three-predicate subgroup searches for every dependency outcome.

Iteration 1 tested the clearest oncogene-like main effects. Four strong binary-feature signals were supported. Cell lines with feature_009 == 1 had much stronger BRAF dependency (mean dependency_BRAF -0.805 versus -0.250; effect -0.555; p < 1e-300). feature_022 == 1 marked stronger EGFR dependency (mean -0.601 versus -0.252; effect -0.349; p < 1e-300). feature_024 == 1 marked stronger ERBB2 dependency (mean -0.686 versus -0.253; effect -0.432; p < 1e-300). feature_014 == 1 marked stronger KRAS dependency (mean -0.559 versus -0.252; effect -0.307; p < 1e-300).

Iteration 2 extended the main-effect screen. feature_027 == 1 was associated with a smaller but significant RIT1 dependency (mean -0.353 versus -0.257; effect -0.096). Higher numeric feature_016 was associated with more negative POLR2A dependency (linear slope -0.305). feature_018 == 1 was associated with more negative RPL11 dependency (mean -0.361 versus -0.252; effect -0.109).

Iterations 3 and 4 focused on KIF18A because it showed the strongest heterogeneity. Colorectal lineage (feature_011 == colorectal) was strongly associated with KIF18A dependency (mean -0.720 versus -0.252; effect -0.468). feature_035 == 1 and high feature_028 were also individually associated with more negative KIF18A. The systematic subgroup search found a much stronger complete subgroup: feature_011 == colorectal, feature_035 == 1, and feature_028 >= 0.98 had mean dependency_KIF18A -1.183 versus -0.253 in all other lines (effect -0.930). Adding feature_020 clarified the subgroup: when feature_020 == 0, the subgroup had mean -1.379 versus -0.253 (effect -1.126), while the same colorectal/feature_035/high-feature_028 subgroup with feature_020 == 1 had mean about -0.253 and therefore did not show enhanced KIF18A dependency. This supports feature_020 == 0 as an enabling predicate and feature_020 == 1 as suppressing the KIF18A dependency.

Iterations 5 and 6 ran dependency-specific heterogeneity searches for BRAF, EGFR, ERBB2, KRAS, and RIT1. BRAF remained best explained by feature_009 == 1. EGFR was dominated by feature_022 == 1, with the strongest small subgroup found at feature_022 == 1, feature_016 >= 0.964, and feature_011 == brain (mean -0.658 versus -0.278; effect -0.381). ERBB2 remained best explained by feature_024 == 1. KRAS remained best explained by feature_014 == 1. RIT1 remained a weaker dependency marked primarily by feature_027 == 1.

Iterations 7 through 9 covered the remaining dependency outcomes. TMED10 did not show a large oncogene-like subgroup, but feature_016 >= 0.999, feature_026 >= 1.08, and feature_010 == luminal had modestly more negative dependency_TMED10 (mean -0.338 versus -0.254; effect -0.084; p = 0.00034). The relatively resistant TMED10 subgroup feature_016 <= 0.66, feature_026 <= 0.919, and feature_027 == 1 had mean -0.166 versus -0.254 (effect +0.089; p = 0.00032). DHX9 was modestly more negative in feature_016 >= 0.964, feature_023 >= 5.669, and feature_026 >= 1.101 lines (mean -0.400 versus -0.333; effect -0.067; p = 7.18e-7). POLR2A showed stronger quantitative-feature heterogeneity, especially feature_016 >= 0.999, feature_010 == adenocarcinoma, and feature_003 >= 0.428 (mean -0.558 versus -0.418; effect -0.139; p = 2.3e-13). RPL11 was more negative in feature_003 >= 0.548, feature_016 >= 0.964, and feature_023 >= 5.85 lines (mean -0.409 versus -0.266; effect -0.142).

Overall conclusions: the clearest dependency-specific signals are BRAF-feature_009, EGFR-feature_022, ERBB2-feature_024, KRAS-feature_014, and the much richer KIF18A subgroup. KIF18A is the best example of interaction-driven dependency: colorectal context, feature_035 == 1, high feature_028, and feature_020 == 0 together define a far stronger signal than any single feature. TMED10 and DHX9 show only modest subgroup effects. POLR2A and RPL11 appear tied to broader quantitative feature states rather than one isolated binary driver. The feature_005 batch labels did not emerge as the explanation for the main dependency signals.
