Analysis Summary - Dataset ds001_breast (n = 50,000 patients, outcome = pfs_months)
====================================================================================

Cohort PFS distribution: mean 4.69, median 4.54, range 0.00-16.66 months. 129 columns
(126 anonymized features + patient_id + outcome). Three columns are categorical strings:
patient_id (unique), feature_011 (race: white/asian/black/hispanic/other), feature_089
(insurance: medicaid/private/medicare/uninsured). Of the 126 anonymized features,
~82 are binary, ~7 are small-cardinality ordinal (e.g., 0-2, 0-4, 0-10), and the
remainder are continuous integer or float.

ITERATION-BY-ITERATION FINDINGS
-------------------------------

Iteration 1 - Strongest single-feature signals (univariate)
- feature_080 (continuous, range 30-90, mean 65) is a powerful POSITIVE prognostic
  factor. Pearson r = +0.70, slope = +0.176 months PFS per unit. Quintile means rise
  monotonically from 2.28 (Q1) to 7.21 (Q5). It alone explains R^2 = 0.49 of PFS
  variance.
- feature_063 (ordinal 0/1/2) is a strong NEGATIVE prognostic factor. Mean PFS
  drops from 5.64 (=0) to 4.44 (=1) to 3.29 (=2). Linear slope -1.18/unit, r = -0.32.
- feature_056 (binary, prevalence 30%) is a strong NEGATIVE prognostic factor:
  PFS 3.62 vs 5.15, diff = -1.54 months (p ~ 0).

Iteration 2 - Additional main effects
- feature_042 (binary, prev 35%): +1.10 months when present (p ~ 0).
- feature_048 (binary, prev 10%): -1.05 months when present (p = 1.5e-178).
- feature_111 (binary, prev 70%): +0.57 months (p = 4e-143).
- feature_039 (binary, prev 64%): +0.36 months (p = 6e-59).
- feature_015 (binary, prev 35%): -0.56 months.
- feature_067 continuous: r = -0.12.
- feature_019 continuous: r = +0.10. feature_101 continuous: r = -0.09.

Iteration 3 - Demographic main effects (RACE, INSURANCE)
- Race (feature_011): one-way ANOVA F = 0.85, p = 0.49. PFS means cluster tightly
  around 4.7 across all 5 categories. Black-vs-white pairwise: diff = +0.04, p = 0.29.
  No race effect on PFS.
- Insurance (feature_089): ANOVA F = 0.31, p = 0.82. Medicaid 4.71, private 4.69,
  medicare 4.69, uninsured 4.73. Medicaid-vs-private: diff = +0.02, p = 0.54.
  No insurance effect on PFS.
- feature_080 relationship is approximately linear (no threshold or U-shape).

Iteration 4 - Multivariable model (no interactions yet)
- A 13-feature OLS achieves R^2 = 0.79. Independent coefficients (all p<0.001 unless
  noted): feature_080 +0.176, feature_063 -1.17, feature_056 -1.55, feature_042 +1.10,
  feature_048 -0.98, feature_111 +0.53, feature_019 +0.48, feature_040 -0.47,
  feature_015 -0.58, feature_067 -0.077, feature_101 -0.019.
- feature_039 (p=0.32) and feature_034 (p=0.88) are NOT independent after
  adjustment; their univariate effects are confounded by stronger correlates.
- feature_080 x feature_063 interaction: -0.0057/unit (p=6e-08). Statistically
  significant but trivially small relative to main effects -- the prognostic effects
  are essentially additive on the additive scale.

Iteration 5 - Predictive interaction screening
- Pairwise screen of plausible "treatment" binary features (feature_111, feature_042,
  feature_039, feature_028, feature_068, feature_020) crossed with potential
  "biomarker" binary features (feature_015, feature_040, feature_022, feature_014,
  feature_076, feature_048, feature_056) revealed a striking pattern:
  - feature_042 x feature_015: main +1.66, interaction -1.62 (p = 4e-255)
  - feature_042 x feature_040: main +1.35, interaction -1.36 (p = 2e-118)
  - feature_111 x feature_015: main +0.85, interaction -0.80 (p = 1e-56)
  - feature_111 x feature_040: main +0.69, interaction -0.66 (p = 4e-26)
  - feature_039 x feature_015: main +0.55, interaction -0.56 (p = 1e-30)
  - feature_039 x feature_040: main +0.44, interaction -0.42 (p = 3e-12)
  In each case the interaction term is the negative mirror image of the main
  effect, indicating QUALITATIVE interaction: the treatment benefit exists when
  the biomarker is 0 and is fully erased when the biomarker is 1.

Iteration 6 - Stratified confirmation
- feature_042 stratified by feature_015: in feature_015=0, diff = +1.66 (p ~ 0);
  in feature_015=1, diff = +0.04 (p = 0.29). Effect entirely abolished.
- feature_042 stratified by feature_040: feature_040=0 diff = +1.35 (p ~ 0);
  feature_040=1 diff = -0.01 (p = 0.77). Effect abolished.
- feature_015 and feature_040 are statistically INDEPENDENT (chi^2 p = 0.35), so
  they capture two separate dimensions of treatment resistance.

Iteration 7 - Continuous and ordinal effect modifiers (negative tests)
- feature_080 x feature_042 interaction: -0.0032/unit (p = 0.044). Effect of
  feature_042 across feature_080 quartiles is +1.16/+1.12/+1.07/+1.04 -- essentially
  uniform; no clinically meaningful modification.
- feature_063 x feature_042 interaction: +0.009 (p = 0.78). feature_042 effect by
  performance-status-like stratum 0/1/2 = +1.12/+1.09/+1.15. No modification.
- The treatment effect of feature_042 therefore depends only on the binary
  biomarkers feature_015 and feature_040, not on continuous prognostic covariates.

Iteration 8 - Joint biomarker subgroup and demographic disparity tests
- Joint feature_015 x feature_040 strata:
    (0,0) n=26760 -- feature_042 effect = +2.04 months (p ~ 0)
    (0,1) n=5875  -- feature_042 effect = -0.03 (p = 0.56)
    (1,0) n=14180 -- feature_042 effect = +0.04 (p = 0.29)
    (1,1) n=3185  -- feature_042 effect = +0.02 (p = 0.78)
  ONLY the doubly-negative cohort benefits, and the benefit is roughly twice as
  large there as in the marginal estimate (+1.10) because the marginal estimate
  averages across non-responders.
- feature_042 effect by RACE: white +1.107, asian +1.062, black +1.060,
  hispanic +1.115, other +1.007. All highly significant; range 1.01-1.12.
  No racial disparity in treatment benefit.
- feature_042 effect by INSURANCE: medicaid +1.081, private +1.091,
  medicare +1.110, uninsured +1.097. No insurance-related disparity.

Iteration 9 - Multivariable models within biomarker subgroups
- Doubly-negative subgroup (n=26760): independent coefs feature_042 +2.05,
  feature_111 +0.98, feature_039 +0.034 (p = 0.17, NS once feature_111 is
  adjusted), feature_080 +0.176, feature_063 -1.18, feature_056 -1.54,
  feature_048 -0.97. R^2 = 0.73. feature_039 is therefore a confounded marker
  rather than an independent third treatment; its univariate signal arises because
  feature_039 is highly correlated with feature_111 (cross-tab confirms most
  feature_111=1 patients are also feature_039=1).
- Biomarker-positive subgroup (any of feature_015=1 or feature_040=1, n=23240):
  feature_042 +0.0015 (p=0.80), feature_111 +0.013 (p=0.11), feature_039 -0.004
  (p=0.66) -- ALL three "treatment" signals are completely null. Prognostic
  factors (feature_080, feature_063, feature_056, feature_048) are unchanged
  in magnitude. R^2 = 0.96 in this subgroup because removing the treatment-effect
  variance leaves an almost-deterministic prognostic structure.

Iteration 10 - Synergy and prognostic-effect homogeneity
- In the doubly-negative subgroup, fitting feature_042 + feature_111 +
  feature_042*feature_111 (with covariates) gives:
    feature_042 alone (when feature_111=0): 0.0000 (p = 0.999)
    feature_111 alone (when feature_042=0): 0.0003 (p = 0.989)
    feature_042 x feature_111 interaction: +2.917 (p ~ 0)
  Confirmed by the 4-cell mean-PFS table inside the doubly-negative subgroup:
    feature_042=0, feature_111=0 -> 4.31 months (n=5324)
    feature_042=0, feature_111=1 -> 4.30 months (n=12110)
    feature_042=1, feature_111=0 -> 4.27 months (n=2755)
    feature_042=1, feature_111=1 -> 7.21 months (n=6571)
  Neither feature_042 nor feature_111 alone produces ANY benefit; the two together
  produce a +2.9-month gain. This is a strong SYNERGISTIC interaction -- both
  treatments must be co-administered (and both biomarkers must be negative) for
  any benefit to occur.
- feature_080's slope is identical in all four (feature_015 x feature_040) cells:
  +0.1753, +0.1772, +0.1772, +0.1774. It is a purely prognostic factor that
  operates equivalently across all biomarker statuses.

OVERALL CONCLUSIONS
-------------------

1. Prognostic structure (biomarker-agnostic). The dataset's PFS variance is
   dominated by a small set of biomarker-INDEPENDENT prognostic factors:
   feature_080 (continuous, +0.176/unit, R^2 ~0.49 by itself), feature_063
   (ordinal 0/1/2, -1.18/unit), feature_056 (-1.54), feature_048 (-0.98). These
   features have nearly identical effects regardless of treatment or biomarker
   status. feature_080 alone explains roughly half of all PFS variance and the
   four prognostic features together explain ~95% of variance in the
   biomarker-positive subgroup, where treatment effects are null.

2. Predictive structure (treatment-by-biomarker heterogeneity). Three nominally
   "active" features (feature_042, feature_111, feature_039) display large
   univariate positive associations with PFS, but each one's effect is fully
   ABOLISHED by either feature_015=1 or feature_040=1 -- two independent binary
   resistance markers. Stratified analyses confirm: in patients with EITHER
   resistance marker positive, no treatment effect remains (p>0.1 for all three).
   feature_039 is not an independent third treatment but is a co-marker with
   feature_111 (its univariate signal vanishes once feature_111 is adjusted for).

3. Synergy. Within the eligible (doubly-negative) population, feature_042 and
   feature_111 are SYNERGISTIC: neither produces any benefit alone, but in
   combination they yield ~+2.9 months of additional PFS. This suggests both
   treatments must be co-administered to a patient whose biomarkers are both
   negative for any treatment effect to manifest.

4. No demographic disparities. Race (feature_011) and insurance status
   (feature_089) have NO main effect on PFS (ANOVA p = 0.49 and 0.82
   respectively) and do NOT modify the feature_042 treatment effect (effect
   size 1.01-1.12 across all 5 race categories and 4 insurance categories).
   In this synthetic-looking cohort, treatment benefit is allocated strictly
   by biomarker status, not by patient demographics.

5. Hypothesis verdicts.
   Supported (significant, direction as predicted): h1, h2, h3, h4, h5, h6,
     h7, h8, h9, h10, h11, h14, h15, h16 (statistical only), h17, h18, h19,
     h20, h21, h22, h23, h27, h28, h29, h31 (null treatment effects in
     biomarker-positive cohort), h32, h33.
   Refuted (no significant effect found, consistent with null): h12, h12b,
     h13, h13b, h24 (feature_015 and feature_040 are independent, supporting
     the predicted null association), h25, h26, h30 (feature_039 is NOT an
     independent treatment after adjustment).

Bottom line: The dataset implements a clean canonical oncology pattern -- strong
prognostic factors that operate uniformly, two independent qualitative
biomarkers of treatment resistance, and a synergistic two-drug regimen whose
effect manifests only when both biomarkers are wild-type. Race and insurance
play no role.
