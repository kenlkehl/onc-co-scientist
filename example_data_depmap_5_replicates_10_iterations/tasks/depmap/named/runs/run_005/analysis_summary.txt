Dataset ds001_depmap contains 50000 cancer cell-line CRISPR dependency records. I analyzed the ten dependency outcomes using mean comparisons, continuous-feature association tests, and a systematic heterogeneity search over single-feature masks, quartile-threshold masks for continuous variables, and pairwise subgroups. For the strongest heterogeneous signal, I refined the search into a complete multi-feature subgroup.

Iteration 1 tested BRAF dependency. `braf_v600e == 1` was the dominant signal: `dependency_BRAF` averaged -0.8047 in mutant lines versus -0.2495 in non-mutant lines, an effect of -0.5552 (p < 1e-300). Skin lineage also showed more negative BRAF dependency (effect -0.1783, p < 1e-300), consistent with enrichment for BRAF-driven models.

Iteration 2 tested receptor-amplification dependencies. `egfr_amplification == 1` was associated with more negative `dependency_EGFR` (mean -0.6010 vs -0.2524; effect -0.3486; p < 1e-300). `erbb2_amplification == 1` similarly predicted more negative `dependency_ERBB2` (mean -0.6856 vs -0.2535; effect -0.4321; p < 1e-300).

Iteration 3 tested KRAS dependency. `kras_mutation == 1` strongly predicted more negative `dependency_KRAS` (mean -0.5594 vs -0.2520; effect -0.3075; p < 1e-300). Pancreatic lineage was also more KRAS-dependent than other lineages (effect -0.1127; p < 1e-300).

Iteration 4 tested an NF1/RAS-pathway hypothesis for RIT1. `nf1_loss == 1` lines had more negative `dependency_RIT1` than `nf1_loss == 0` lines (mean -0.3527 vs -0.2572; effect -0.0955; p < 1e-300).

Iterations 5 and 6 identified the strongest novel subgroup. `dependency_KIF18A` was more negative in colorectal lines (effect -0.4680), APC-mutant lines (effect -0.2921), and high-WNT lines with `wnt_activity_score >= 0.819` (effect -0.2168), all with p < 1e-300. Refinement showed the complete subgroup `lineage == "colorectal"`, `apc_mutation == 1`, `wnt_activity_score >= 0.819`, and `smad4_loss == 0` had a very strong KIF18A dependency: mean -1.2990 versus -0.2530 elsewhere, effect -1.0460 (p < 1e-300). The same colorectal/APC-mutant/high-WNT context with `smad4_loss == 1` did not show this dependency (mean -0.2557), indicating that SMAD4 loss suppresses or disables the dependency signal.

Iteration 7 tested MYC-linked ribosomal stress. `myc_amplification == 1` predicted more negative `dependency_RPL11` (effect -0.1091; p < 1e-300), as did high `copy_number_8q24 >= 0.428` (effect -0.0988; p < 1e-300). The combined MYC-amplified/high-8q24 subgroup remained negative (mean -0.3647 vs -0.2524; effect -0.1123; p < 1e-300).

Iteration 8 tested proliferation-linked dependencies. Fast-growing lines with `growth_rate_doublings_per_day >= 0.964` had more negative `dependency_POLR2A` (effect -0.1043; p < 1e-300) and a smaller but significant shift in `dependency_TMED10` (effect -0.0369; p < 1e-300). TMED10 did not show a stronger molecular subgroup than the growth-rate signal in the subgroup scan.

Iteration 9 tested DHX9. High `rnaseq_MYC_log2_tpm >= 5.669` was associated with more negative `dependency_DHX9` (effect -0.0286; p = 2.0e-15), and continuous growth rate was also negatively associated with DHX9 dependency (slope -0.1086 dependency-score units per additional doubling per day; p < 1e-300).

Iteration 10 consolidated the systematic heterogeneity search for each dependency outcome. The leading subgroups were the expected driver lesions for BRAF, EGFR, ERBB2, KRAS, and RIT1; the refined colorectal/APC/high-WNT/SMAD4-intact subgroup for KIF18A; fast growth for POLR2A and TMED10; MYC expression/growth for DHX9; and MYC amplification/high 8q24 copy number for RPL11. Overall, the clearest complex dependency pattern is KIF18A: the effect is not merely colorectal lineage, APC mutation, or WNT activity alone, but their combination with intact SMAD4.
