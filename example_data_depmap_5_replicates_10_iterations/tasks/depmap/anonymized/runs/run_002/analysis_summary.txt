I analyzed the 50,000-row CRISPR dependency map using only the local task bundle. More negative dependency scores were treated as stronger dependency. Tests were Welch two-sample comparisons for subgroup hypotheses and univariate slopes for continuous features; p-values are two-sided normal approximations, with extremely small values capped at 1e-300. I also ran a systematic dependency-heterogeneity search over single predicates, all two-feature predicate pairs, and top three-feature subgroup refinements for each dependency outcome.

The clearest main-effect dependencies were straightforward. `dependency_BRAF` was much more negative in `feature_009` == 1 lines (-0.805 vs -0.250; effect -0.555; p <= 1e-300). `dependency_EGFR` was more negative in `feature_022` == 1 lines (-0.601 vs -0.252; effect -0.349; p <= 1e-300). `dependency_ERBB2` tracked `feature_024` == 1 (-0.686 vs -0.253; effect -0.432; p <= 1e-300), and `dependency_KRAS` tracked `feature_014` == 1 (-0.559 vs -0.252; effect -0.308; p <= 1e-300). The pairwise heterogeneity search found subgroups such as each driver-positive group with `feature_021` == 0, but these did not materially improve the effect beyond the driver feature itself.

`dependency_RIT1` showed a weaker but significant signal: `feature_027` == 1 lines averaged -0.353 versus -0.257 in other lines (effect -0.096; p=2.03e-70). The best pair subgroup, `feature_027` == 1 with `feature_022` == 0, had a similar effect (-0.098; p=2.82e-69), so I interpret the RIT1 result mainly as a `feature_027` association with only modest heterogeneity.

Several outcomes shared a continuous `feature_016` pattern. Higher `feature_016` predicted more negative `dependency_POLR2A` most strongly (slope -0.3048 per unit; p <= 1e-300), with weaker but significant slopes for `dependency_TMED10` (-0.1012; p=1.11e-38) and `dependency_DHX9` (-0.1086; p=2.73e-44). The heterogeneity screen refined these into high-`feature_016` subgroups: `feature_016` >= 0.964 and `feature_027` == 0 for POLR2A (effect -0.105; p=3.34e-175), `feature_016` >= 0.964 and `feature_024` == 0 for TMED10 (effect -0.038; p=4.71e-24), and `feature_016` >= 0.964 and `feature_009` == 0 for DHX9 (effect -0.038; p=3.39e-24). These refinements were statistically significant but smaller than the kinase-like driver effects.

`dependency_RPL11` was associated with both high `feature_003` and `feature_018` == 1. The continuous slope for `feature_003` was -0.1015 per unit (p=8.39e-263), while `feature_018` == 1 lines averaged -0.361 versus -0.252 in other lines (effect -0.109; p=7.04e-144). The combined subgroup `feature_018` == 1 and `feature_003` >= 0.428 averaged -0.365 versus -0.252 in other lines (effect -0.112; p=3.53e-148), so the combined subgroup was supported but not dramatically stronger than the two component signals.

The most biologically distinctive multivariable pattern was for `dependency_KIF18A`. Single-feature tests showed stronger dependency in `feature_011` == 'colorectal' lines (effect -0.468), `feature_035` == 1 lines (effect -0.292), and high-`feature_028` lines (effect -0.217), all with p <= 1e-300. Pair refinements were stronger: colorectal plus `feature_035` == 1 had effect -0.690, and colorectal plus high `feature_028` had effect -0.676. The strongest complete subgroup was `feature_011` == 'colorectal', `feature_035` == 1, and `feature_028` >= 0.819: n=3133, mean `dependency_KIF18A` -1.112 versus -0.253 in all other lines, effect -0.859, p <= 1e-300. This supports a genuine dependency-heterogeneity hypothesis rather than a simple main effect alone.

Overall, the supported hypotheses are strong driver-like binary-feature dependencies for BRAF, EGFR, ERBB2, and KRAS; a weaker RIT1 association with `feature_027`; a broad `feature_016` axis affecting POLR2A/TMED10/DHX9; a modest RPL11 subgroup involving `feature_018` and `feature_003`; and a sharply defined KIF18A subgroup requiring colorectal status, `feature_035` == 1, and high `feature_028`. The main refutation from the heterogeneity search is that most non-KIF18A outcomes did not require complex subgroup definitions beyond their primary driver or continuous feature axis.
