Analysis summary - ds001_nsclc
================================

Cohort: 50,000 NSCLC patients, 33 de-identified features, single outcome
pfs_months (mean 3.43, range 0-15.3, no missing).

Across 17 iterations of propose-test-refine, I identified one dominant
treatment-effect-heterogeneity structure plus a clean set of additive
prognostic factors. Key findings, in order of evidence:

1. PROGNOSTIC STRUCTURE (additive, treatment-independent)
---------------------------------------------------------
A multivariable OLS with main effects only achieved R^2 = 0.94. Adding
the four three-way feature_016 interactions described below raised R^2
to 0.975. The dominant additive prognostic factors are:

  feature_015  +0.168 / unit (continuous, range 30-90).  Pearson r with
               PFS = +0.757 - by far the strongest single predictor.
               Mean PFS rises from 1.14 mo in lowest quintile to 5.92 mo
               in highest quintile. Effect is invariant across treatment
               and biomarker subgroups, so this is a pure prognostic.
  feature_014  -1.10 per ordinal level (0/1/2). Means 4.33 / 3.19 / 2.15.
               Consistent with performance status.
  feature_023  -1.41 (binary, ~65% prevalent). Strong negative.
  feature_011  -0.91 (binary, ~25%). Negative.
  feature_017  -0.073 / unit (continuous, mean 3.84). Modest negative.
  feature_001  former smoker +0.55, never +0.55 vs current (reference).
  feature_006  squamous -0.73 vs adenocarcinoma.
  feature_025  +0.46 / unit (continuous). Modest positive.
  feature_032  -4e-4 / unit (continuous, large numeric range). Tiny.

  Continuous features feature_019, feature_024, feature_010, feature_003,
  feature_030, feature_008, feature_004, feature_013, feature_026,
  feature_029, feature_007, feature_033, feature_009 all have |r| with PFS
  below 0.02 - effectively noise.

2. TREATMENT-EFFECT HETEROGENEITY: feature_016 IS A PREDICTIVE-MARKER-DRIVEN TREATMENT
--------------------------------------------------------------------------------------
Univariate, feature_016 (binary, 12.7% prevalent) is the only positively-
signed binary feature on PFS (+0.78, p=5e-151). Treatment assignment is
balanced across all candidate biomarkers (P(feature_016=1) is ~12.7% in
every biomarker stratum), so the effect heterogeneity below cannot be
attributed to confounding.

A two-way interaction screen across all 13 binaries identified a single
dominant interaction: feature_016 * feature_018 = +2.56 (p=0). Stratified:

  feature_018 = 0:   feature_016 effect = -0.005  (p=0.80, null)
  feature_018 = 1:   feature_016 effect = +2.44   (p<<1e-300)

Within feature_018=1, a second large modifier appeared: feature_031
(binary, ~45% prevalent) with interaction coefficient -1.69 (p=3e-178).
The cross-tab of (feature_018, feature_031) shows feature_016 benefit is
exclusively in (feature_018=1, feature_031=0); the other three quadrants
each have a small NEGATIVE feature_016 effect of around -0.10 to -0.17.

  (f018=0, f031=0): f016 effect = -0.10
  (f018=0, f031=1): f016 effect = -0.14
  (f018=1, f031=0): f016 effect = +4.56  ***
  (f018=1, f031=1): f016 effect = -0.17

Within the (feature_018=1, feature_031=0) responder candidate, two rare
binary features further gate the response:

  feature_028 = 1 (rare, ~5%):  feature_016 effect = -0.29 (n=44 vs 422)
  feature_028 = 0:              feature_016 effect = +4.74 (n=1191 vs 8105)
  feature_005 = 1 (rare, ~3%):  feature_016 effect = +0.08 (n=37 vs 246)
  feature_005 = 0:              feature_016 effect = +4.71 (n=1198 vs 8281)

Both feature_028=1 and feature_005=1 fully abolish the treatment benefit.
A multivariable OLS with three-way interactions confirms each modifier
quantitatively cancels the base benefit:

  feature_016 * feature_018                = +4.77  (p=0)
  feature_016 * feature_018 * feature_031  = -4.59  (p=0)
  feature_016 * feature_018 * feature_028  = -2.40  (p=0)
  feature_016 * feature_018 * feature_005  = -2.77  (p=0)

The three negative three-way terms each independently consume the +4.77
benefit when their respective marker is set, leaving net effect ~0.

The four modifiers (feature_018, feature_031, feature_028, feature_005)
are also striking in another way: among UNTREATED (feature_016=0)
patients, feature_018, feature_031, and feature_005 have NO main effect
on PFS (diffs of -0.005, +0.011, -0.015; all p>0.5). They are PURELY
PREDICTIVE markers of feature_016 response. Only feature_028 carries a
modest standalone prognostic effect (-0.66 in untreated, p=2e-47).

3. FINAL SUBGROUP HYPOTHESIS
-----------------------------
The complete responder subgroup is defined by ALL FOUR predicates:

  feature_018 = 1  AND  feature_031 = 0  AND  feature_028 = 0  AND  feature_005 = 0

  Within subgroup (n=9013, treated 1154):
    Mean PFS, treated = 8.23
    Mean PFS, untreated = 3.35
    Difference = +4.89 months  (t=73.0, p=0.0)
    Response rate (PFS > 6 mo): 83.4% treated vs 10.8% untreated
                                 (chi^2=3302, p=0.0)

  Outside subgroup (n=40987):
    feature_016 effect = -0.128  (p=4e-5; sign opposite, magnitude tiny)

The effect estimate sharpens monotonically as predicates are added:
  All patients                           +0.78
  feature_018=1                           +2.44
  feature_018=1, feature_031=0            +4.56
  feature_018=1, feature_031=0, f028=0    +4.74
  Complete subgroup                       +4.89

4. NEGATIVE FINDINGS / ALTERNATIVES RULED OUT
---------------------------------------------
- No second treatment is present in the dataset. Pairwise binary x binary
  interactions excluding feature_016 yield only modest cross-effects that
  are explained by the underlying f016 responder structure.
- feature_022 (next candidate by univariate sign) has a small +0.27
  effect when feature_018=0 and ~null when feature_018=1, and is null
  in multivariable adjustment (coef -0.006). It does not show a
  precision-medicine signature.
- A regression-tree subgroup search on outcome alone fails to recover
  the predictive markers, exactly because feature_018, feature_031, and
  feature_005 have no main effect: they can only be discovered via
  treatment-by-feature interaction tests.
- Continuous features (other than feature_015 and feature_017) carry no
  detectable signal on PFS and do not modify treatment effect.

5. INTERPRETATION (CLINICAL FRAMING)
-------------------------------------
The pattern - a single binary treatment with strict biomarker-gated
benefit, a positive predictive marker (feature_018) that has no
prognostic value of its own, plus a negative-modifier marker
(feature_031) that switches off the response, plus two rare
secondary-resistance markers (feature_028, feature_005) - is the
canonical signature of a precision-oncology / targeted-therapy regimen
in NSCLC (e.g., a TKI requiring an activating alteration, neutralized
by sex or a co-mutation, and rendered ineffective by rare resistance
biomarkers). The effect is robust, large, and unambiguous in this
50,000-patient cohort.

End of summary.
