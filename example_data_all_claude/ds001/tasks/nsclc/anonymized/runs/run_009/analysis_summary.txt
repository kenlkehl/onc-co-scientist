Analysis summary — ds001_nsclc (n = 50,000; outcome pfs_months)
================================================================

Cohort and outcome
------------------
The dataset has 50,000 NSCLC patients, 33 anonymised features, and a single continuous outcome
pfs_months (range 0.00–15.33, mean 3.43, median 3.32, SD 2.23). 7.3% of patients have pfs_months = 0.
Only 6 patients have pfs ≥ 14 (no censoring cap), so pfs_months was treated throughout as a
continuous outcome and analysed by OLS / Spearman correlation / stratified t-tests rather than by
Kaplan–Meier or Cox regression.

Step 1 — Univariate screen (iterations 1–4)
-------------------------------------------
Among 13 binary features, six showed strong univariate effects (p < 1e-50):
  - feature_023 = 1: −1.44 mo (n = 32,365)
  - feature_011 = 1: −0.92 mo (n = 12,584)
  - feature_016 = 1: +0.78 mo (n =  6,368)
  - feature_028 = 1: −0.81 mo (n =  2,464)
  - feature_018 = 1: +0.32 mo (n = 17,544)
  - feature_031 = 1: −0.20 mo (n = 22,371)

Among continuous features, feature_015 (range 30–90, mean 65) had a Spearman r = +0.769 with
pfs_months — by far the dominant continuous prognostic factor (slope +0.170 mo per unit). feature_017
(r = −0.125) and feature_025 (r = +0.100) were the next strongest. All other continuous features had
|r| < 0.02.

Categorical features:
  - feature_006: adenocarcinoma 3.67 mo vs squamous 2.81 mo (p ~ 0).
  - feature_001 (smoking): never 3.67, former 3.61, current 2.99 (p < 1e-186).
  - feature_014 (3-level ordinal): 0 → 1 → 2 monotonically reduces pfs (4.33 → 3.19 → 2.15;
    F = 3141, p ~ 0). Most consistent with a performance-status- / stage-like ordered prognostic.

Step 2 — Multivariable adjustment (iteration 5)
-----------------------------------------------
A multivariable OLS adjusting for feature_015, feature_023, feature_011, feature_014, hist_squamous,
smoking_current, smoking_never, feature_017, feature_025, feature_031, feature_016, feature_018 (and
checks of feature_021, feature_022, feature_028) achieved R² = 0.894. After adjustment:
  - feature_022, feature_028, feature_021 lost significance — confounded by stronger covariates,
    not robust independent contributors.
  - feature_016 (+0.89, p ~ 0) and feature_018 (+0.32, p ~ 0) retained the largest positive
    independent effects, identifying them as the leading treatment candidates.
  - Negative independent effects were confirmed for feature_023 (−1.42), feature_014 (−1.10 / level),
    feature_011 (−0.91), squamous (−0.72), current smoking (−0.55), feature_017 (−0.07/unit) and
    feature_031 (−0.20).

Step 3 — Treatment-effect heterogeneity (iterations 7–17)
---------------------------------------------------------
Treatment-by-feature interaction screens for the two candidate treatments (feature_016 and
feature_018) revealed three modifiers that abolished or reversed each candidate’s benefit:
feature_031, feature_028, and feature_005.

Stratified estimates of feature_016:
  - feature_031 = 0: +1.54 mo (p = 5e-301); feature_031 = 1: −0.15 mo (p = 3e-04, opposite direction).
  - feature_028 = 0: +0.82 mo (p = 7e-162); feature_028 = 1: −0.39 mo (p = 0.008).
  - feature_005 = 0: +0.81 mo (p = 3e-155); feature_005 = 1: −0.01 mo (p = 0.95).

In the joint subgroup feature_031 = 0 ∧ feature_028 = 0 ∧ feature_005 = 0 (“eligible”),
feature_016 effect was +1.65 mo, while in every other (feature_031, feature_028, feature_005)
cell it was between −0.53 and +0.23 mo.

Stratified estimates of feature_018 followed the same pattern (effect concentrated in eligible
patients) plus a smoking gradient (current +0.51, former +0.30, never 0.01 mo).

Step 4 — Combination synergy (iterations 12–15)
-----------------------------------------------
The single largest interaction in the entire dataset is feature_016 × feature_018 (coef +2.53,
p ~ 0, adjusted). Joint means make this stark:
  - feature_016 = 0, feature_018 = 0 → 3.33 mo (n = 28,334)
  - feature_016 = 0, feature_018 = 1 → 3.33 mo (n = 15,298)
  - feature_016 = 1, feature_018 = 0 → 3.21 mo (n =  4,122)
  - feature_016 = 1, feature_018 = 1 → 5.77 mo (n =  2,246)

Either feature alone has essentially no effect; only the combination is active. Once a
feature_016 × feature_018 interaction is in the model, the individual main effects vanish
(both p > 0.7), confirming the apparent univariate “positive main effects” of these two
features were entirely driven by the small sub-population receiving both.

Three-way stratification of the combination effect by feature_031 × feature_028 × feature_005
(iteration 14) shows that the combination benefit is essentially confined to one cell:
  - feature_031 = 0 ∧ feature_028 = 0 ∧ feature_005 = 0: +4.89 mo (p ~ 0)
  - feature_031 = 0 ∧ feature_028 = 0 ∧ feature_005 = 1: +0.25 mo (p = 0.49)
  - feature_031 = 0 ∧ feature_028 = 1 ∧ feature_005 = 0: −0.36 mo (p = 0.30)
  - feature_031 = 1 ∧ feature_028 = 0 ∧ feature_005 = 0: −0.13 mo (p = 0.06)
  - feature_031 = 1 ∧ feature_028 = 1 ∧ feature_005 = 0: −0.80 mo (p = 0.026)

A confirmatory adjusted model (iteration 15) with combo_eligible = feature_016·feature_018·eligible
achieves R² = 0.9785 and yields combo_eligible = +4.92 mo (95% CI 4.90–4.94), with both feature_016
and feature_018 main coefficients indistinguishable from zero — the combination benefit is fully
localised to the eligible subgroup.

Step 5 — Robustness within eligible (iteration 16–17)
-----------------------------------------------------
Within eligible patients, the combination effect is homogeneous across other binary covariates
(stratum-specific point estimates 4.7–5.1 mo for feature_023, feature_011, feature_022, feature_021,
hist_squamous, and smoking, except for the very small never-smoker eligible-combo cell n = 21).
Within eligible patients, the combination effect rises modestly with feature_015 (interaction
+0.013 mo per unit, p = 2e-40), but the combination is positive across the entire feature_015
range. No second eligibility-defining variable surfaced.

Step 6 — Other features (iterations 18–24)
------------------------------------------
  - feature_023, feature_011: prognostic only. Negative effect persisted in every stratum tested
    and the magnitude (−0.9 to −1.4 mo) was small relative to combo (+4.9 mo).
  - feature_022, feature_021, feature_028 (as marker, not modifier): minor / confounded prognostic
    contributions.
  - feature_015 dose-response (slope ≈ +0.17 mo per unit, R² of univariate fit ≈ 0.57) is the single
    largest contributor to outcome variance and is independent of the treatment story.
  - feature_025 (continuous, range 1.7–5.5) is a positive prognostic continuous variable
    (+0.46 mo/unit) but does not modify the treatment effect.

A comprehensive binary × binary interaction screen (iteration 22) confirmed that the only
substantively large interactions in the data involve the cluster {feature_016, feature_018,
feature_031, feature_028, feature_005}; no second independent treatment combination exists.

Final treatment-effect subgroup (iteration 25)
----------------------------------------------
The best-supported treatment-by-biomarker hypothesis for the only outcome with usable
treatment variation is:

  In NSCLC patients with feature_031 = 0 AND feature_028 = 0 AND feature_005 = 0,
  receipt of the combination feature_016 = 1 AND feature_018 = 1 is associated with
  approximately +4.9 months longer pfs_months than any other combination of feature_016
  and feature_018 status (adjusted; p ~ 0). In patients with feature_031 = 1, feature_028 = 1,
  or feature_005 = 1 the combination provides no benefit on pfs_months.

Hypothesis support and refutation
---------------------------------
  Supported:
    h1 (pfs continuous, no censoring); h2/h3/h4 (univariate screens identified named features);
    h5 (multivariable independent prognostics); h6 (feature_016 / feature_018 as treatments);
    h7–h11 (heterogeneity of feature_016 and feature_018 by feature_031 / feature_028 / feature_005);
    h12 (feature_016 × feature_018 synergy is real and large; individual main effects disappear);
    h13–h15 (combination effect localised to feature_031 = 0 ∧ feature_028 = 0 ∧ feature_005 = 0);
    h16–h17 (effect homogeneous across other binary modifiers within eligible; mild +feature_015 boost);
    h18–h21 (feature_023, feature_011, feature_022, feature_021 are prognostic, not treatments);
    h22 (no additional treatment combination); h23 (feature_015 dominant continuous prognostic);
    h24 (feature_025 prognostic, not modifier); h25 (final subgroup).

  Refuted / refined:
    Initial impression that feature_016 and feature_018 each individually benefit pfs (refuted —
      neither has a main effect once their interaction is modelled).
    Initial impression that feature_022, feature_028 (as main effect), and feature_021 are
      prognostic markers independent of the dominant prognostic set (refuted — they lose
      significance after adjustment).

Overall conclusions
-------------------
1. pfs_months in this 50,000-patient NSCLC cohort is dominated by a small set of prognostic
   factors — feature_015, feature_023, feature_011, feature_014, feature_017, feature_025,
   histology (squamous), smoking (current), and feature_031 — together explaining ~89% of
   variance in pfs_months.
2. There is exactly one robust treatment-effect signal in the data: the combination of
   feature_016 = 1 AND feature_018 = 1, which adds ≈ +4.9 mo to pfs_months. Neither feature
   alone has any effect.
3. The combination benefit is restricted to the “eligible” subgroup defined by feature_031 = 0
   AND feature_028 = 0 AND feature_005 = 0. Each of these three markers is a sufficient
   condition for non-response: presence of any one of them (≈ 51% of the cohort, with
   feature_031 = 1 alone capturing 45%) drops the combination effect to ≈ 0 mo. No further
   binary modifier substantively expands or contracts the eligible-combo effect.
4. feature_023, feature_011, feature_014, hist_squamous, current smoking, feature_017, and
   feature_031 are unfavourable prognostic factors. feature_015 and feature_025 are favourable
   prognostic continuous covariates. None of these is a treatment-effect modifier of comparable
   magnitude to the {feature_031, feature_028, feature_005} eligibility set.
