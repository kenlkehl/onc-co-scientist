Analysis Summary — ds001_prostate (n = 50000)
=================================================

Outcome
-------
The cohort has a single binary outcome, `objective_response`, with 24.0%
prevalence (12011 / 50000 events). One feature (`feature_030`) is constant
and was excluded.

The 32 features partition into 12 binary indicators, two ordinal
multi-level features (`feature_001` with 3 levels, `feature_010` with 5
levels — likely a Gleason-style score), and 17 continuous variables.
Plausible clinical identities (based on ranges and means) include
`feature_016` ≈ age (mean 65), `feature_022` ≈ PSA (range 0.12–3622, heavy
right tail), `feature_026` ≈ hemoglobin (mean 12.5).

Univariate associations
-----------------------
Eight features showed strong univariate associations with response. The
single positive predictor was `feature_008` (binary, prevalence 40%,
log-odds +1.10, OR ≈ 3.0, p ≪ 1e-300). All other strongly significant
binary features were negatively associated: `feature_013` (-1.07),
`feature_015` (-0.62), `feature_021` (-0.64), `feature_027` (-0.40), and
the ordinal `feature_001` (-0.29 per level). Continuous PSA-like
`feature_022` was negatively associated (-0.005 per unit, p = 1.6e-94).

Multivariable model
-------------------
A single multivariable logistic regression confirmed the same set of
independent predictors: `feature_008` (+1.20), `feature_013` (-1.13),
`feature_015` (-0.69), `feature_021` (-0.74), `feature_027` (-0.43),
`feature_001` (-0.33 per level), `feature_020` (-0.14 per SD),
`feature_022` (-0.10 per SD), `feature_024` (-0.07 per SD), and
`feature_002` (+0.06 per SD). The remaining binary features (006, 005, 011,
023, 017, 019, 004) and most continuous lab values had small, non-significant
adjusted effects.

The pattern — exactly one binary feature with a strong positive adjusted
effect on a tumor-response endpoint — is consistent with `feature_008`
being a therapeutic indicator (e.g., an active anti-cancer agent), with
the negative-effect features representing adverse disease/host factors
(metastatic burden, prior progression, etc.).

Treatment-effect heterogeneity
------------------------------
The bulk of the analysis was a systematic search for heterogeneity of the
`feature_008` treatment effect on `objective_response`.

Per-feature interaction screen against `feature_008`. The strongest
interactions were all negative:
  • `feature_013`:  -1.98 (p ≪ 1e-300)
  • `feature_015`:  -1.24 (p = 1.4e-93)
  • `feature_022`:  -0.81 per SD (p = 1.2e-72)
  • `feature_021`:  -1.14 (p = 3.3e-42)
  • `feature_027`:  -1.28 (p = 1.8e-19)
  • `feature_001`:  +0.14 per level (p = 1.6e-5; small positive)

Each of the four binary modifiers, when set to 1, abolishes the treatment
effect. We then enumerated the 16 joint binary cells of (013, 015, 021,
027). The result is a sharp cliff: in the (0,0,0,0) cell (n = 15681,
31% of the cohort) the response rate jumps from 17.2% (untreated) to 79.8%
(treated) — a 62.6 percentage-point absolute benefit (OR ≈ 19). In every
one of the other 15 cells the absolute treatment effect is between -12 and
+9 percentage points, and on average ≈ 0.

Subgroup definition
-------------------
We define the "favorable" subgroup as

    feature_013 == 0  AND  feature_015 == 0  AND
    feature_021 == 0  AND  feature_027 == 0

A formal logistic model with treatment, favorable-indicator, and the
treatment × favorable interaction estimates a near-null residual T main
effect (+0.06, p = 0.05, OR 1.06) and a very large positive interaction
(+2.89, p ≪ 1e-300). Within the favorable subgroup the implied treatment
log-odds is +2.94 (OR ≈ 19); after multivariable adjustment for the other
27 features the treatment OR within the favorable subgroup is ≈ 20.2.

Necessity. Each of the four excluders is necessary. Removing them
one-at-a-time gives within-subgroup absolute treatment-effect differences:

  • drop  013=0  →  +0.287   (largest dilution; 013 is the most important)
  • drop  015=0  →  +0.505
  • drop  021=0  →  +0.563
  • drop  027=0  →  +0.607   (smallest, but still meaningful)
  • full def     →  +0.626

Adding excluders is monotonic. Single-excluder definitions yield diffs
of +0.21–0.44; two-excluder defs +0.25–0.55; three-excluder +0.29–0.61;
the full four-excluder def is the cleanest.

Within-favorable robustness. Inside the favorable subgroup we searched
for further modifiers and found essentially none:
  • Other binary features (006, 005, 023, 011, 017, 019, 004): T-effect
    differences across strata all within ±0.04 of +0.62.
  • `feature_001` levels 0/1/2: +0.65/+0.62/+0.59 (small monotone fall).
  • `feature_010` levels 6–10: +0.63/+0.62/+0.64/+0.62/+0.63 (flat).
  • `feature_022` (PSA-like): coefficient on response per SD is -0.082
    in the T=0 arm and -0.092 in the T=1 arm — i.e., a prognostic effect
    of the same magnitude in both arms. The earlier feature_008 ×
    feature_022 interaction is therefore confounded by correlation between
    `feature_022` and the four binary excluders, not a true effect modifier
    inside the favorable subgroup.

Within-non-favorable check. Inside the complement subgroup (any of the
four excluders = 1) the overall treatment effect is +0.008 (essentially
zero). Interaction screens of treatment with every other feature inside
the non-favorable subgroup yielded only small effects (top interaction was
`feature_005` × T at p = 0.004, well above Bonferroni). There is no
identifiable sub-subgroup in which feature_008 produces a benefit when any
of the four excluders is positive.

Other binary features as treatment candidates. We re-ran the interaction
screen with each of the seven other binary features in turn. None has a
strong adjusted main effect, and none shows the magnitude of treatment-effect
heterogeneity that `feature_008` shows. `feature_008` is the unique
treatment-like variable in the dataset.

Confounding-by-indication check. Treatment prevalence is 40.3% in the
favorable subgroup and 40.1% in the non-favorable subgroup — the
heterogeneity is not produced by selection on the favorable indicator.

Other prognostic factors. Independent prognostic effects (response-lowering
unless noted) include: `feature_001` (-0.33/level, p = 1e-82), `feature_020`
(-0.14/SD, p = 2e-33), `feature_022` (-0.10/SD, p = 3e-8), `feature_024`
(-0.07/SD, p = 9e-8), `feature_002` (+0.06/SD, p = 2e-8). Inside the
favorable subgroup, these features remain prognostic but not predictive
(similar coefficients in T=0 and T=1).

Hypotheses supported vs refuted
-------------------------------
Supported:
  • feature_008 is an active treatment with a large positive effect.
  • feature_013, feature_015, feature_021, feature_027 are individually
    independent negative prognostic factors AND each abolishes the
    feature_008 treatment effect.
  • feature_001 and feature_022 are independent negative prognostic
    factors.
  • The joint subgroup definition (all four excluders = 0) is necessary
    and sufficient: every excluder set to 1 eliminates the benefit, and no
    other feature is needed inside the favorable subgroup.

Refuted / null:
  • Other binary features (006, 005, 023, 011, 017, 019, 004) are NOT
    treatment indicators (no main effects, no interaction-modifying
    behavior).
  • Within the favorable subgroup, feature_022 is NOT a treatment-effect
    modifier (the apparent main interaction is confounded with the four
    binary excluders).
  • Within the non-favorable subgroup, no sub-subgroup recovers a
    feature_008 benefit.

Final treatment-effect subgroup conclusion
------------------------------------------
Patients treated with feature_008 have substantially higher probability of
objective_response than untreated patients IF AND ONLY IF

    feature_013 == 0  AND  feature_015 == 0  AND
    feature_021 == 0  AND  feature_027 == 0.

Within this subgroup (n = 15681, 31% of cohort): response 79.8% vs 17.2%
(absolute benefit +62.6 percentage points, OR ≈ 20). Outside this subgroup
(n = 34319): response 16.1% vs 15.3% (absolute benefit +0.8 percentage
points, OR ≈ 1.06; not clinically or statistically meaningful). The
treatment-effect heterogeneity is the dominant signal in the dataset.
