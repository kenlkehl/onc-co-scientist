I then tested whether pairs of features showed subgroup heterogeneity or interaction on the response-probability scale. Binary variables used 1 vs 0, while multi-valued variables used above-median vs at-or-below-median groups for adequate cell sizes. For each pair, the effect estimate was the four-cell difference-in-differences: response(high,high) - response(high,low) - response(low,high) + response(low,low). Positive values mean the joint high/present subgroup responded more than expected from the separate feature effects; negative values mean less than expected. Overall conclusion: objective_response in this cohort is not explained by one feature alone. Several feature columns show reproducible main-effect differences in absolute response probability, and several feature pairs show clinically meaningful heterogeneity on the same natural probability scale. The most persuasive findings are the associations and interactions that remain significant after false-discovery adjustment and have nontrivial absolute risk differences; nominal-only or near-zero effects in the transcript should be treated as weak or refuted hypotheses.

Caveats: feature labels are de-identified opaque variables, so the results describe statistical treatment-biomarker-outcome structure but cannot be clinically interpreted as named biomarkers without a separate data dictionary. All tests are observational associations, not causal estimates.
