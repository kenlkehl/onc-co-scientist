Oncology cohort ds001_aml analysis summary

The local parquet dataset contained 50000 AML patient records, 75 opaque feature columns, and the binary outcome objective_response. The overall objective_response rate was 18.75%. Because feature labels are de-identified, all conclusions are statistical associations with the named feature columns rather than clinical claims about known treatments or biomarkers.

Interpretation:
The evidence supported several large binary-feature associations with objective_response in both favorable and unfavorable directions. The strongest associations persisted after adjustment for other top-ranked binary features, suggesting they were not merely duplicates of a single dominant correlate. Continuous features had statistically detectable but smaller per-SD effects; top-quartile contrasts were useful for describing magnitude but were generally less dominant than binary main effects. Interaction testing found candidate subgroup heterogeneity among the strongest binary features, with the most pronounced pair showing a materially different response-rate contrast across subgroups. Several weak binary features were also tested and did not support a response association, which helps separate strong signals from background variation.
