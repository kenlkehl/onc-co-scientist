Dataset ds001_prostate contained 50000 patients, 72 de-identified feature columns, and the binary outcome objective_response. The overall objective_response rate was 0.240. All analyses treated effects on the response-probability scale; binary features were tested as value 1 versus 0, and continuous or multilevel features were tested per one standard-deviation increase.

Main-effect exploration found several leading univariate patterns and a few weaker directional checks:
- feature_005: objective_response 0.361 for value 1 vs 0.159 for value 0; risk difference 0.202, p=<1e-300.
- feature_034: objective_response 0.154 for value 1 vs 0.346 for value 0; risk difference -0.192, p=<1e-300.
- feature_027: per-1-SD risk difference 0.009, p=4.35e-6.
- feature_009: objective_response 0.160 for value 1 vs 0.260 for value 0; risk difference -0.101, p=1.04e-122.
- feature_018: per-1-SD risk difference 0.004, p=0.0537.
- feature_003: per-1-SD risk difference -0.037, p=1.21e-85.
- feature_053: objective_response 0.246 for value 1 vs 0.238 for value 0; risk difference 0.008, p=0.0918.
- feature_065: objective_response 0.150 for value 1 vs 0.250 for value 0; risk difference -0.101, p=3.94e-76.

Multivariable refinement used a heteroskedasticity-robust linear probability model including the ten strongest univariate predictors. The leading adjusted effects were:
- feature_005: adjusted effect 0.202 (robust SE 0.004, p=<1e-300)
- feature_034: adjusted effect -0.188 (robust SE 0.004, p=<1e-300)
- feature_009: adjusted effect -0.100 (robust SE 0.004, p=1.01e-116)
- feature_023: adjusted effect -0.034 (robust SE 0.002, p=1.07e-85)
- feature_065: adjusted effect -0.102 (robust SE 0.006, p=4.28e-71)
- feature_063: adjusted effect -0.022 (robust SE 0.002, p=1.59e-35)
- feature_003: adjusted effect -0.009 (robust SE 0.001, p=2.17e-9)
- feature_055: adjusted effect -0.063 (robust SE 0.010, p=2.46e-9)

The adjusted model showed that several top univariate signals were not merely duplicates of one another, although magnitudes changed after conditioning on the other leading predictors. Effects should still be interpreted as exploratory because features are de-identified and many hypotheses were screened.

Subgroup and interaction probing found the following strongest supported heterogeneity signals among tested leading pairs:
- feature_034 x feature_005: -0.433 (p=<1e-300)
- feature_009 x feature_005: -0.243 (p=2.20e-168)
- feature_034 x feature_009: 0.221 (p=3.01e-146)
- feature_065 x feature_005: -0.218 (p=2.76e-85)
- feature_003 x feature_005: -0.061 (p=5.78e-19)
- feature_003 x feature_009: 0.032 (p=5.22e-18)
- feature_003 x feature_065: 0.034 (p=3.93e-13)
- feature_003 x feature_055: 0.034 (p=3.90e-4)

Negative or weak checks helped bound the search; these features showed little evidence of association with objective_response in the tested form:
- feature_037: effect -0.002, p=0.2173.
- feature_070: effect 0.002, p=0.2313.

Overall, the cohort shows a concentrated response signal: a small set of opaque features, especially the strongest positive and negative main-effect columns listed above, carried most of the objective_response variation. Interaction analyses suggested that some binary and mixed feature pairs modify each other's response associations on the additive risk scale, supporting subgroup heterogeneity beyond isolated main effects. Because the feature names are anonymized, these results identify treatment-biomarker-outcome style patterns statistically but cannot assign clinical semantics to the features without a data dictionary.
