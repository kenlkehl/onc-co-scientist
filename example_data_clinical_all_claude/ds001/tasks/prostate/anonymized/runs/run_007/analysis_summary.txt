Analysis Summary -- Dataset ds001_prostate (N = 50,000)
========================================================

Outcome
-------
The single outcome `objective_response` is binary; 12,011 / 50,000 patients (24.0%)
respond. All 32 candidate predictors are anonymized as `feature_001` ... `feature_032`.

Approach
--------
Across 25 iterations I (1) screened every feature for a univariate main effect,
(2) fit a multivariable logistic model to identify independently associated
predictors, (3) classified the binary features into "candidate treatments" vs
"prognostic / disease markers" by their marginal effect and their correlation
with other features, and (4) ran a systematic treatment-by-feature interaction
screen for every candidate treatment, including a hand-built joint subgroup
analysis and a tree-based check for the strongest treatment.

Main effects (supported)
------------------------
The strongest predictors of `objective_response`:

* feature_008 (binary, prevalence 40%) -- POSITIVE: response 36.1% vs 15.9%
  off, +20.2 absolute percentage points (chi-square p ~ 0; OR 2.99). This is
  the largest positive predictor in the cohort.

* feature_013 (binary, 55%) -- NEGATIVE: response 15.4% vs 34.6%, -19.3
  pp (OR 0.34, p ~ 0). feature_013 turns out to be approximately a high-vs-low
  dichotomization of the continuous feature_022 (KS = 0.51; mean feature_022
  is 12 when feature_013 = 0 and 68 when feature_013 = 1).

* feature_022 (continuous, 0.12 - 3622, highly skewed) -- NEGATIVE; raw r =
  -0.087 (p ~ 3e-85). Log-transformed it explains substantially more variance
  (pseudo-R^2 0.044 vs 0.013).

* feature_015 (binary, 20%) -- NEGATIVE: 16.0% vs 26.0% (OR 0.54, p ~ 2e-98).

* feature_021 (binary, 10%) -- NEGATIVE: 15.0% vs 25.0% (OR 0.53, p ~ 5e-56).

* feature_027 (binary, 3%) -- NEGATIVE: 17.6% vs 24.2% (OR 0.67, p ~ 3e-9).

* feature_001 (ordinal 0/1/2) -- NEGATIVE: 28.2% / 22.8% / 18.2% across the
  three levels (p ~ 2e-73).

* feature_002 (continuous, 1.5 - 5.5) -- POSITIVE; small but significant
  (delta +0.024, p ~ 4e-6); robust in multivariable model (beta = +0.13).

* feature_020 (continuous, 0 - 24) -- NEGATIVE; mean 3.5 (responders) vs 3.95
  (non), p ~ 3e-29; multivariable beta = -0.036.

* feature_024 (continuous) -- NEGATIVE; small effect (delta -0.53, p ~ 5e-8).

Refuted main effects
--------------------
Many features show no association with `objective_response` even at
N = 50,000:

* All-zero / constant: feature_030 (single value, dropped).

* Continuous: feature_016 (likely age 30-90), feature_010 (range 6-10,
  consistent with Gleason score), feature_018, feature_026, feature_007,
  feature_032, feature_014, feature_028, feature_009, feature_029, feature_003,
  feature_012, feature_025, feature_031 -- all p > 0.10 univariately.

* Binary with no main effect: feature_006, feature_005, feature_011,
  feature_017, feature_019, feature_023, feature_004 -- all |delta| < 0.005,
  p > 0.30. These were tested as candidate "randomized treatments with HTE
  only".

Treatment-effect heterogeneity (the central finding)
----------------------------------------------------
feature_008 behaves like a treatment indicator whose effect is dramatically
modified by four binary features (and the closely linked continuous feature_022):

* feature_008 x feature_013 (interaction p ~ 0): when feature_013 = 0 the
  treatment raises response from 16.9% to 61.0% (delta +44 pp, OR 7.7).
  When feature_013 = 1 the treatment effect is essentially nil:
  15.0% -> 15.8% (delta +0.8 pp, p = 0.08).

* feature_008 x feature_022 (interaction p ~ 1e-72): treatment effect by
  feature_022 quartile is +36.7, +26.0, +14.8, +3.4 pp -- a monotone
  attenuation that mirrors the feature_013 finding (since feature_013 ~
  high feature_022).

* feature_008 x feature_015 (p ~ 1e-93): in feature_015 = 1 patients the
  treatment effect drops to ~+1 pp even among feature_013 = 0 patients
  (17.7% vs 16.3%). feature_015 independently abolishes the effect.

* feature_008 x feature_021 (p ~ 3e-42): in feature_021 = 0, treatment
  delta is +22.5 pp; in feature_021 = 1, delta is +0.6 pp.

* feature_008 x feature_027 (p ~ 2e-19): treatment delta +20.9 pp when
  feature_027 = 0, delta -2.2 pp when feature_027 = 1.

* feature_008 x feature_001 (p ~ 1.6e-5 on the logit scale): on the absolute
  scale the effect is roughly constant (+20 pp at every level), so the
  interaction is mainly relative-risk -- minor clinical heterogeneity.

A pre-specified joint search across the seven other binary candidates
(feature_005, feature_006, feature_011, feature_017, feature_019, feature_023,
feature_004) found NO interaction surviving a Bonferroni-adjusted threshold
across 217 tests; smallest p was 0.010. Hence feature_008 is the only
treatment-like variable in this cohort.

A depth-4 decision tree fit to the entire feature set rediscovers the same
structure: split 1 on feature_013, split 2 on feature_008 in the
feature_013 = 0 branch, then on feature_015 / feature_021. The leaf
"feature_013=0 & feature_008=1 & feature_015=0 & feature_021=0" has
predicted response near 80%.

Best-supported subgroup hypothesis (final)
------------------------------------------
For the outcome `objective_response`, the treatment feature_008 produces a
strong positive effect almost exclusively in the subgroup defined by

    feature_013 = 0  AND  feature_015 = 0  AND  feature_021 = 0  AND  feature_027 = 0

In that subgroup (N = 15,681, 31% of the cohort):
    response rate with feature_008 = 1: 0.798
    response rate with feature_008 = 0: 0.172
    absolute treatment effect:           +0.626
    odds ratio:                          18.98 (p ~ 0)

Outside that subgroup (N = 34,319):
    treatment effect:                    +0.007 (p > 0.05)

Adding the further restriction feature_001 = 0 (N = 5,511) raises the
absolute treatment effect to +0.646 (0.859 vs 0.213, OR 22.4) -- a modest
additional enrichment.

Interpretation
--------------
A simple rule -- "predicted responder = feature_008 = 1 AND feature_013 = 0
AND feature_015 = 0 AND feature_021 = 0 AND feature_027 = 0" -- has a
positive predictive value of 0.80 (sensitivity 0.43, specificity 0.97) for
objective response. This pattern is consistent with an oncology drug whose
benefit is confined to patients without high values on a key disease-burden
biomarker (feature_022 / feature_013) and without three additional
unfavorable disease markers; the four binary markers function approximately
as a coarse summary of an underlying refractory-disease axis. The remaining
seven binary features look like balanced covariates (or noise) without
predictive value, and most laboratory-style continuous features
(feature_007, feature_026, feature_014, feature_032, etc.) carry no signal
in this cohort.

(h1 trivially, h9-h10 and h18 confirmed as null, h27-h30, h34) are
