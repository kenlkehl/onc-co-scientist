ANALYSIS SUMMARY — ds001_prostate (n = 50,000)
===============================================

Cohort: 50,000 prostate-cancer patients (sex_female = 0 for every record). One
binary outcome: objective_response (overall rate 24.0%). Six treatment
indicators (enzalutamide, abiraterone, docetaxel, olaparib, lu177_psma,
pembrolizumab). Treatments are not mutually exclusive — 21% of patients
received none, 40% one, 28% two, 9% three, 1.5% four, ≈0.1% five, and one
patient all six. All adjusted analyses below use simultaneous logistic
regression with the full set of features and all six treatment indicators
as covariates.

----------------------------------------------------------------
1. PROGNOSTIC MAIN EFFECTS ON objective_response
----------------------------------------------------------------
Strong negative prognostic factors (chi-square or univariate logistic
p < 0.001 AND adjusted p < 0.001):

  • mcrpc=1                  : -19.2 pp absolute response
                               (15.4% vs 34.6%); adjusted logit -1.13
  • brca2_mutation=1         : -10.1 pp; adjusted logit -0.74
  • ar_v7_positive=1         : -10.1 pp; adjusted logit -0.69
  • msi_high=1               :  -6.6 pp; adjusted logit -0.43
  • higher ecog_ps           : adjusted logit -0.33 per unit
  • higher psa_ng_ml         : adjusted logit -0.001 per ng/mL
                               (lowest PSA quartile 31.8% vs highest 15.3%)
  • higher weight_loss_pct_6mo: adjusted logit -0.036 per percent
  • higher crp_mg_l          : adjusted logit -0.007 per mg/L

Positive prognostic factor:
  • higher albumin_g_dl      : adjusted logit +0.13 per g/dL

No detectable main effect on response: visceral_mets, psma_high,
gleason_score, age_years, ldh_u_l, nlr, hemoglobin_g_dl,
alkaline_phosphatase_u_l, ast_u_l, alt_u_l, total_bilirubin_mg_dl,
creatinine_mg_dl, bun_mg_dl, sodium_meq_l, potassium_meq_l, calcium_mg_dl.

----------------------------------------------------------------
2. UNADJUSTED AND ADJUSTED TREATMENT MAIN EFFECTS
----------------------------------------------------------------
Unadjusted on-vs-off response-rate differences:

  enzalutamide   : +20.2 pp  (36.1% vs 15.9%)
  abiraterone    :  -0.3 pp
  docetaxel      :  -0.0 pp
  olaparib       :  -0.1 pp
  lu177_psma     :  +0.3 pp
  pembrolizumab  :  -0.2 pp

Full multivariable logistic regression (all features + all treatments):

  treatment_enzalutamide  +1.20 logit  (p ≈ 0)         <-- only meaningful main effect
  treatment_abiraterone   -0.025 logit (p = 0.31)
  treatment_docetaxel     +0.002 logit (p = 0.95)
  treatment_olaparib      +0.026 logit (p = 0.49)
  treatment_lu177_psma    +0.029 logit (p = 0.35)
  treatment_pembrolizumab +0.014 logit (p = 0.79)

Hypothesis tested and supported: only enzalutamide has a positive adjusted
main effect on objective_response.

----------------------------------------------------------------
3. SYSTEMATIC TREATMENT × FEATURE INTERACTION SCREEN
----------------------------------------------------------------
For each of 6 treatments × 25 features (150 tests), an unadjusted logistic
treatment + feature + treatment*feature was fit. Top interactions all
involve enzalutamide:

  enza × mcrpc           : β_int = -1.98  p ≈ 0
  enza × ar_v7_positive  : β_int = -1.24  p = 1.4e-93
  enza × psa_ng_ml       : β_int = -0.009 p = 1.2e-72
  enza × brca2_mutation  : β_int = -1.14  p = 3.3e-42
  enza × msi_high        : β_int = -1.28  p = 1.8e-19
  enza × ecog_ps         : β_int = +0.14  p = 1.6e-5  (smaller benefit at high ECOG)

For non-enzalutamide treatments, no single interaction reached p < 1e-2 in
the screen. The few weak signals (each p between 0.01 and 0.05) were:

  lu177_psma × visceral_mets  : β_int = -0.19  p = 0.010
  docetaxel × sodium_meq_l    : β_int = -0.017 p = 0.023
  abiraterone × brca2_mutation: β_int = +0.19  p = 0.031
  olaparib × bun_mg_dl        : β_int = -0.015 p = 0.031
  pembrolizumab × gleason     : β_int = +0.10  p = 0.033
  olaparib × brca2_mutation   : β_int = -0.29  p = 0.045

----------------------------------------------------------------
4. CLASSIC BIOMARKER → THERAPY MATCHUPS — ALL EXPECTED PAIRINGS
   FAIL OR REVERSE
----------------------------------------------------------------
Each of the canonical "biomarker-targeted" pairings was tested by
stratifying within the biomarker-positive subgroup:

  • pembrolizumab in msi_high=1 (n=1,528)
       on=17.7% vs off=17.6%, diff +0.1 pp, p=1.0  → REFUTED (no benefit)
  • olaparib in brca2_mutation=1 (n=4,996)
       on=12.1% vs off=15.3%, diff -3.2 pp, p=0.058
       Adjusted interaction p=0.054, sign negative → REVERSED direction
  • lu177_psma in psma_high=1 (n=29,962)
       on=23.8% vs off=23.9%, diff -0.1 pp, p=0.88 → REFUTED
  • abiraterone in ar_v7_positive=1 (n=10,038)
       on=16.3% vs off=15.8%, diff +0.5 pp, p=0.55 → no effect
  • abiraterone in mcrpc=1 (n=27,481)
       on=15.4% vs off=15.3%, diff +0.1 pp, p=0.89 → no effect
  • docetaxel in mcrpc=1 (n=27,481)
       on=15.2% vs off=15.4%, diff -0.1 pp, p=0.77 → no effect
  • docetaxel in visceral_mets=1 (n=9,973)
       on=24.2% vs off=23.8%, diff +0.4 pp, p=0.68 → no effect

This is the central surprise of the dataset: classic biology-textbook
pairings (PARP/BRCA2, anti-PD1/MSI, PSMA-targeted/PSMA-high) do not yield
the expected enriched response. Olaparib actually trends opposite to
expectation in BRCA2-mutated patients.

----------------------------------------------------------------
5. ENZALUTAMIDE: A FOUR-PREDICATE RESPONSIVE SUBGROUP
----------------------------------------------------------------
Iterative refinement (stratifying within mcrpc=0, then ar_v7=0, then brca2=0,
then msi=0) demonstrated that the enzalutamide effect is fully
concentrated in the conjunction of four binary "absent" markers:

    Responsive subgroup S* :  mcrpc = 0
                              AND ar_v7_positive = 0
                              AND brca2_mutation = 0
                              AND msi_high = 0
                              (n = 15,681; 31.4% of cohort)

Within S*:    enzalutamide on  79.8%  (n=6,325)
              enzalutamide off 17.2%  (n=9,356)
              difference        +62.6 pp absolute   (p ≈ 0)

Outside S*:   enzalutamide on  16.1%
              enzalutamide off 15.3%
              difference         +0.8 pp            (p = 0.06)

Adjusted (full covariates) treatment × subgroup interaction = +2.94 logit
(p ≈ 0); the adjusted enzalutamide effect when in S* is +3.01 logit.

NECESSITY of each predicate: dropping any one of the four predicates from
the conjunction (i.e., requiring the other three but the violated fourth)
collapses the +63 pp benefit to ≤ 2 pp:

  Drop mcrpc=0     → diff +0.7 pp (n=19,163)
  Drop ar_v7=0     → diff +2.2 pp (n=3,940)
  Drop brca2=0     → diff +0.4 pp (n=1,775)
  Drop msi=0       → diff -2.5 pp (n=494)

The four predicates are jointly necessary AND sufficient. PSMA-high and
visceral-mets do NOT modify the enzalutamide effect. Within S*, ECOG and
PSA continue to be prognostic for response but do NOT modify the
enzalutamide effect: the on-vs-off difference is +63.3 / +62.8 / +62.4 /
+61.8 pp across PSA quartiles and +64.6 / +62.2 / +59.0 across ECOG 0/1/2.
None of the other five treatments adds incremental benefit within S*
(each on-vs-off difference within ±2 pp).

----------------------------------------------------------------
6. PEMBROLIZUMAB: A LOW-INFLAMMATION RESPONSIVE SUBGROUP
----------------------------------------------------------------
Exhaustive 2-way and 3-way binary subgroup search and continuous-feature
interaction analysis converged on the same definition:

    Pembrolizumab-responsive subgroup S† :
        albumin_g_dl ≥ 3.8 (population median)
        AND
        crp_mg_l    ≤ 3.34 (population median)
        (n = 13,678; 27.4% of cohort)

Within S†:    pembro on  31.5%  (n=670)
              pembro off 25.3%  (n=13,008)
              difference  +6.2 pp   (chi-square p = 4.4e-4)

Outside S†:   pembro on  20.9%
              pembro off 23.5%
              difference  -2.7 pp   (chi-square p = 0.012, sign reversed)

Adjusted (full covariates) treatment × subgroup interaction = +0.44 logit
(p = 1.1e-4); adjusted pembro effect when in S† = +0.32 logit.

BOTH predicates are necessary (single-predicate "loose" definitions yield
NEGATIVE pembrolizumab effects):

  alb_high alone (with crp high) : diff -2.3 pp
  crp_low alone (with alb low)   : diff -2.9 pp
  both bad (alb low AND crp high): diff -2.8 pp
  inflam_good (both)             : diff +6.2 pp

Within S†, the pembrolizumab effect is reasonably uniform across ECOG,
mCRPC, AR-V7, BRCA2, MSI-high, PSMA-high, and visceral mets substrata
(stratified diffs +2.6 to +13.0 pp). The pembrolizumab benefit is also
preserved within the enzalutamide-non-responsive subgroup (n=9,475 in
inflam_good=1 ∩ enza-non-responsive: diff +5.4 pp).

----------------------------------------------------------------
7. OLAPARIB, LU177-PSMA, ABIRATERONE, DOCETAXEL — NO POSITIVE
   RESPONSIVE SUBGROUP
----------------------------------------------------------------
For each remaining treatment, a gradient-boosting S-learner CATE estimate
yielded a near-degenerate distribution at zero, with no top-decile
subgroup showing >0.3 pp on-vs-off difference. Targeted adjusted
treatment × feature interaction tests confirmed:

  • Olaparib: adjusted main effect coef = +0.046 (p = 0.23). The only
    notable interaction is olaparib × brca2_mutation = -0.29 (p = 0.054),
    with an UNFAVORABLE direction (BRCA2-mutated patients respond worse on
    olaparib than off it: 12.1% vs 15.3%, diff -3.2 pp). No subgroup with
    a positive olaparib effect was identified.

  • Lu177-PSMA: adjusted main effect coef = +0.029 (p = 0.35). Adjusted
    lu177 × visceral_mets interaction = -0.21 (p = 0.0098). Stratified:
    visceral_mets = 0 → +1.0 pp (n=40,027, p = 0.10);
    visceral_mets = 1 → -2.4 pp (n=9,973,  p = 0.043). Effect magnitude
    too small to define a clinically meaningful responsive subgroup.

  • Abiraterone: adjusted main effect coef = -0.025 (p = 0.31). Adjusted
    abi × brca2_mutation = +0.19 (p = 0.045). Stratified: brca2 = 0 →
    -0.5 pp; brca2 = 1 → +2.1 pp (p = 0.06). Too small to define a
    clinically meaningful responsive subgroup.

  • Docetaxel: adjusted main effect coef = +0.002 (p = 0.95). All
    docetaxel × feature interactions tested have p > 0.15 in adjusted
    models; no subgroup of n > 1,000 with on-vs-off difference exceeding
    ±5 pp survived cross-cut replication. Docetaxel is inert in this
    cohort.

----------------------------------------------------------------
8. OVERALL CONCLUSIONS
----------------------------------------------------------------
(i) Of six treatments in the dataset, exactly two carry a clear,
    biomarker-defined response signal:
        — Enzalutamide produces a +63 pp absolute response increase in the
          conjunction subgroup (mcrpc=0 ∧ ar_v7_positive=0 ∧
          brca2_mutation=0 ∧ msi_high=0); each of those four predicates is
          necessary, and the conjunction is sufficient.
        — Pembrolizumab produces a +6 pp absolute response increase in the
          conjunction subgroup (albumin_g_dl ≥ median 3.8 ∧ crp_mg_l ≤
          median 3.34); both predicates are necessary.

(ii) Olaparib, lu177-PSMA, abiraterone, and docetaxel show no responsive
     subgroup with a positive treatment effect detectable from the
     available features. Olaparib's only signal is a marginal negative
     interaction with brca2_mutation; lu177-PSMA's only signal is a
     marginal negative interaction with visceral_mets; abiraterone's only
     signal is a marginal positive interaction with brca2_mutation (small
     magnitude); docetaxel is fully inert.

(iii) Several "textbook" biomarker-targeted pairings (PARP inhibition in
      BRCA2-mutated cancers, anti-PD-1 in MSI-high cancers, PSMA-directed
      radioligand in PSMA-high cancers, AR-targeted therapy in
      AR-V7-positive disease) do NOT yield enriched responses in this
      dataset. Where biology would predict benefit, the dataset shows
      either null or — in the BRCA2/olaparib case — reversed direction.
      This is a striking pattern and may reflect the deliberate construction
      of the synthetic cohort.

(iv) Strong prognostic markers for response (independent of treatment) are
     mcrpc, ar_v7_positive, brca2_mutation, msi_high, ecog_ps, psa_ng_ml,
     weight_loss_pct_6mo, crp_mg_l (negative) and albumin_g_dl (positive).

   definitions, with all necessary predicates)
----------------------------------------------------------------

                subgroup defined by ALL of:
                    mcrpc = 0
                    AND ar_v7_positive = 0
                    AND brca2_mutation = 0
                    AND msi_high = 0
                Direction: + (response rate +62.6 pp absolute).
                p ≈ 0; adjusted t × subgroup interaction p ≈ 0.

H_FINAL_PEMBRO: Pembrolizumab increases objective_response within the
                subgroup defined by ALL of:
                    albumin_g_dl ≥ 3.8
                    AND crp_mg_l ≤ 3.34
                Direction: + (response rate +6.2 pp absolute).
                p = 4.4e-4 (chi-square within subgroup);
                adjusted t × subgroup interaction p = 1.1e-4.

H_FINAL_OLAP  : No responsive subgroup. Marginal negative effect within
                brca2_mutation = 1 (diff -3.2 pp, p = 0.058).

H_FINAL_LU177 : No clinically meaningful responsive subgroup. Marginal
                negative effect within visceral_mets = 1 (diff -2.4 pp,
                p = 0.043).

H_FINAL_ABI   : No clinically meaningful responsive subgroup. Marginal
                positive effect within brca2_mutation = 1 (diff +2.1 pp,
                p = 0.062).

H_FINAL_DOC   : No responsive subgroup detectable from the available
                features.
