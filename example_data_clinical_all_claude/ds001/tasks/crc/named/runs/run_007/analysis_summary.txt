Analysis Summary — ds001_crc (50,000 metastatic CRC patients, single outcome: pfs_months)
======================================================================================

OVERVIEW
--------
Across 25 iterations I started with main effects of patient features and treatments
on progression-free survival (PFS), then carried out a systematic
treatment-effect heterogeneity search via interaction screens, stratified
subgroup tests, joint multivariable models, and exhaustive treatment x feature
scans. The dataset contains six treatments (cetuximab, bevacizumab,
pembrolizumab, encorafenib, trastuzumab/tucatinib, regorafenib) and the usual
panel of CRC biomarkers (KRAS, NRAS, BRAF V600E, MSI-high, HER2-amplified,
NTRK fusion) plus standard clinical and laboratory features.

The core finding is that one treatment — regorafenib — drives essentially all
of the treatment-related variance in PFS, with a benefit that is sharply
restricted to a four-feature joint subgroup. None of the other five treatments
produce detectable PFS effects, including in their canonical biomarker-defined
indication subgroups.

PROGNOSTIC MAIN EFFECTS
-----------------------
Strongly negative for PFS (all p effectively 0 in adjusted models):
  - ECOG performance status: -1.17 months per ECOG point
  - stage_iv: -1.36 months
  - kras_mutation: -0.35 months
  - braf_v600e: -0.36 months
  - right_sided_primary: -0.29 months
  - weight_loss_pct_6mo: -0.08 months per percentage point
  - cea_ng_ml: -0.005 months per ng/mL

Strongly positive for PFS:
  - albumin_g_dl: +0.47 months per g/dL
  - age_years: +0.176 months per year (UNEXPECTED direction; persists after
    full multivariable adjustment for ECOG, stage, biomarkers, labs, and
    treatments — older patients have BETTER PFS in this cohort, with a near-
    monotonic gradient from PFS ~0.3 mo in the 30s to ~7.8 mo in the 80s).

Null or weak prognostic features:
  - sex_female (p=0.77)
  - msi_high, her2_amplified, ntrk_fusion (all NS as main effects)
  - nras_mutation: small +0.22 months univariate (p=0.0002), but this is
    fully explained by mutual exclusivity with kras_mutation; adjusted
    coefficient is +0.06, p=0.32.
  - LDH, CRP, NLR, hemoglobin, alkaline phosphatase, ast, alt, total
    bilirubin, creatinine, BUN, sodium, potassium, calcium are at most
    weakly prognostic.

TREATMENT MAIN EFFECTS
----------------------
Univariate t-tests / OLS on each treatment vs PFS:
  - treatment_regorafenib: ON 5.09 vs OFF 4.12, diff = +0.97 months, p ~ 0
  - treatment_cetuximab: -0.04, p=0.09 (NS)
  - treatment_bevacizumab: -0.02, p=0.35 (NS)
  - treatment_pembrolizumab: +0.005, p=0.86 (NS)
  - treatment_encorafenib: +0.005, p=0.87 (NS)
  - treatment_trastuzumab_tucatinib: -0.04, p=0.25 (NS)

Only regorafenib shows a meaningful main effect.

EXPECTED BIOMARKER-MATCHED INDICATIONS (ALL NULL)
-------------------------------------------------
The dataset does NOT recapitulate the canonical biomarker-treatment matches
of metastatic CRC:
  - pembrolizumab in msi_high=1 (n=2513): adj coef = +0.04, p=0.72
  - encorafenib in braf_v600e=1 (n=2272): adj coef = -0.23, p=0.06 (trends
    negative, not positive)
  - trastuzumab_tucatinib in her2_amplified=1 (n=1504): adj coef = +0.14, p=0.48
  - cetuximab in RAS-wt + BRAF-wt + LEFT (n=16443): adj coef = -0.07, p=0.06
    (also trends slightly negative, NOT a benefit)
  - bevacizumab in braf_v600e=1: appeared negative unadjusted (-0.24, p=0.007)
    but the effect was fully eliminated by adjustment for ECOG, stage, albumin,
    CEA (adj coef +0.003, p=0.85) — a confounding artifact, not a real effect.

The interaction terms for these "expected" biomarker-treatment combinations
are all non-significant (e.g., pembrolizumab x msi_high: p=0.99; encorafenib x
braf_v600e: p=0.38; trastuzumab/tucatinib x her2_amplified: p=0.78).

REGORAFENIB EFFECT-MODIFIER DISCOVERY
-------------------------------------
An exhaustive treatment x feature interaction scan (6 treatments x 27 features)
returned significant interactions (p < 0.001) ONLY for regorafenib, with the
following modifiers:

  - regorafenib x kras_mutation: int coef = -1.629, p = 7.7e-226
      Effect in KRAS-wt: +1.66 months; in KRAS-mut: +0.03 months.
      KRAS mutation almost completely abolishes the treatment effect.
  - regorafenib x right_sided_primary: int coef = -1.454, p = 4.1e-168
      Effect in left-sided: +1.48; in right-sided: +0.03.
      Right-sided primary almost completely abolishes the effect.
  - regorafenib x cea_ng_ml: int coef = -0.0253 per ng/mL, p = 8.7e-58
      Each ng/mL of CEA reduces effect by ~0.025 months.
      Stratified by CEA quartile (overall cohort): Q1 +1.79, Q2 +1.82, Q3
      +0.27, Q4 +0.00. The effect is concentrated in CEA quartiles 1-2
      (CEA below ~5 ng/mL).
  - regorafenib x braf_v600e: int coef = -1.183, p = 6.1e-23
      Effect in BRAF wt: +1.03; in BRAF V600E+: -0.16 (slightly reversed).
  - regorafenib x nras_mutation: +0.988, p = 1.2e-11. NRAS mutation does NOT
    suppress the effect (NRAS-mut patients are by construction KRAS-wt;
    they retain the regorafenib benefit). The relevant exclusion is KRAS, not
    RAS-pathway in general.

No other treatment shows any significant interaction with any feature in the
dataset (continuous or binary).

JOINT REGORAFENIB-RESPONDER SUBGROUP
------------------------------------
Combining the four independent modifiers yields a clean responder definition:

  kras_mutation = 0 AND braf_v600e = 0 AND right_sided_primary = 0 AND cea_ng_ml < 5

In this subgroup (n = 9,326):
  - regorafenib ON (n = 1,903): mean PFS = 9.10 months
  - regorafenib OFF (n = 7,423): mean PFS = 4.09 months
  - Treatment effect: +5.01 months, p ~ 0
  - Multivariable-adjusted (age, sex, ECOG, stage, albumin, CRP, NLR, weight
    loss, MSI, HER2, NRAS, all other treatments): coef = +4.98, p ~ 0.

In the complement (n = 40,674): regorafenib effect = +0.025 months, statistically
detectable only because of the very large n but clinically negligible.

The CEA threshold was identified by a scan within the KRAS-/BRAF-/LEFT
subgroup: thresholds <3, <4, <5 all give effect ~+5.0; effect drops sharply
to ~0 at CEA >= 5. CEA = 5 ng/mL is the conventional clinical upper limit of
normal — a clinically interpretable cutoff.

Each of the four modifiers is necessary. Removing any one (i.e., reverting to a
3-feature subgroup) reduces the effect substantially:
  - KRAS-/BRAF-/LEFT (any CEA): +2.76
  - KRAS-/BRAF-/CEA<5 (any side): markedly attenuated
  - BRAF-/LEFT/CEA<5 (any KRAS): markedly attenuated
  - KRAS-/LEFT/CEA<5 (any BRAF): mildly attenuated (BRAF V600E is rare so
    the four-modifier and three-modifier subgroups overlap heavily)

Within the four-feature responder subgroup, no remaining laboratory or
demographic feature significantly modifies the regorafenib effect (all
p >= 0.07 in interaction tests). The subgroup definition is therefore
considered complete.

NEGATIVE / NULL FINDINGS WORTH NOTING
-------------------------------------
- Treatment x ECOG, x stage, x age, x sex: no significant interaction for any
  treatment (only encorafenib x age_ge65 marginal at p=0.035).
- Co-administration of other treatments with regorafenib does not modify its
  effect within the responder subgroup; effect is +5.0 months whether other
  drugs are coadministered or not.
- The age-PFS positive association (+0.176/yr) is robust, persists in subgroups
  including the regorafenib responder subgroup, and is not explained by
  treatment selection (treatment age distributions are nearly uniform across
  drugs). It represents an unusual feature of the cohort.

FINAL TREATMENT-EFFECT SUBGROUP HYPOTHESIS (PFS)
------------------------------------------------
H58 (final, refined): treatment_regorafenib is associated with substantially
longer pfs_months (approximately +5 months PFS) in metastatic CRC patients
meeting ALL of:
  (1) kras_mutation = 0
  (2) braf_v600e = 0
  (3) right_sided_primary = 0
  (4) cea_ng_ml < 5

Outside this subgroup the regorafenib treatment effect is effectively zero.
No other treatment in the panel shows a significant PFS effect in any subgroup,
including its canonical biomarker-eligible indication subgroup. Most expected
biomarker-treatment matches (pembrolizumab/MSI-high, encorafenib/BRAF V600E,
trastuzumab-tucatinib/HER2 amplified, cetuximab/RAS-wt-LEFT) are absent in
this cohort.
