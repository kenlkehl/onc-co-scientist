ANALYSIS SUMMARY — ds001_breast (n=50,000)
Outcome: pfs_months (continuous, range 0–16.66, mean 4.69, sd 2.50)
Iterations: 25 propose-analyze-refine cycles. 34 hypotheses, 44 statistical analyses.

----------------------------------------------------------------------
1. UNIVARIATE PROGNOSTIC LANDSCAPE (Iterations 1–7)
----------------------------------------------------------------------

Strong univariate associations with pfs_months were found for many features. The
single largest signal is feature_012 (continuous, range 30–90, mean 65),
which shows a near-linear positive relationship with pfs_months: r = +0.702,
slope ≈ +0.176 months per unit, p ≈ 0. The relationship was confirmed to be
linear across quintiles (no meaningful quadratic term), so no subgroup-specific
treatment of this variable is needed. Within strata of every other feature the
direction is preserved.

feature_005 has three ordered levels with monotonically falling pfs_months:
level 0 = 5.64 mo, level 1 = 4.44 mo, level 2 = 3.29 mo. All pairwise
comparisons p ≪ 1e-100. The level-0-to-2 contrast is −2.35 months; the
ordering and stepwise size are consistent with a stage-like covariate.

Other independently prognostic features after multivariable adjustment (OLS
R² = 0.79) — supported direction and magnitude:

  Positive prognostics (higher pfs_months):
    feature_012 +0.176/unit    feature_013 +1.10
    feature_029 +0.53          feature_024 +0.48/unit

  Negative prognostics (lower pfs_months):
    feature_006 −1.55          feature_001 −0.98
    feature_005 level 1 −1.19  feature_005 level 2 −2.34
    feature_028 −0.58          feature_018 −0.47
    feature_004 −0.077/unit    feature_033 −0.019/unit

All p-values for these effects are ≪ 1e-100. By contrast, balanced binary
features feature_030, feature_019, feature_011, and feature_023 collapse to
near-null adjusted main effects (p > 0.08 each), motivating an
interaction-based search.

----------------------------------------------------------------------
2. CANDIDATE TREATMENT IDENTIFICATION (Iteration 8)
----------------------------------------------------------------------

Four binary features had prevalence in the 0.45–0.64 range AND lost their
univariate signal after adjustment, the empirical fingerprint of a treatment
indicator with possibly heterogeneous effects:

  feature_030 prev 0.50,  univariate p = 0.29, adjusted p = 0.08
  feature_019 prev 0.60,  univariate p = 0.31, adjusted p = 0.41
  feature_011 prev 0.45,  univariate p = 0.56, adjusted p = 0.20
  feature_023 prev 0.64,  univariate diff +0.36 mo (p = 5.8e-59)
                          adjusted p = 0.32

feature_023 was unique: it carried a meaningful univariate signal that
disappeared after adjustment — the classical signature of an effect that is
real but heterogeneous across modifiable subgroups.

----------------------------------------------------------------------
3. SYSTEMATIC HETEROGENEITY SEARCH (Iterations 9–17)
----------------------------------------------------------------------

For each candidate treatment we ran an OLS interaction screen — adding
treat × f_i to the fully-adjusted model for every other feature. Results
diverged sharply by candidate.

  feature_030 — minimum interaction p ≈ 0.009 (with feature_012). No
  modifier with biologically meaningful magnitude. Treatment effect is
  flat across all features. NOT a treatment with heterogeneity.

  feature_019 — minimum p ≈ 0.0015 (feature_023), but interaction
  effects all small (|eff| < 0.07 months). Modest, not clinically
  meaningful.

  feature_011 — minimum p ≈ 0.02 (feature_025). Same conclusion: no
  meaningful subgroup signal.

  feature_023 — DRAMATIC heterogeneity. Multiple modifiers had
  p-values that defy chance:
      feature_023 × feature_013 :  +1.029, p ≈ 0
      feature_023 × feature_028 :  −0.557, p = 5e-139
      feature_023 × feature_033 :  −0.017/unit, p = 6e-86
      feature_023 × feature_018 :  −0.416, p = 4e-52

  feature_016 — meaningful but smaller interactions (× feature_013
  +0.38, p = 4e-67). Stratified analysis showed feature_016's
  prognostic effect is itself confined to feature_013==1 patients
  (+0.39 mo there vs essentially null in feature_013==0). Behaves
  more like a prognostic biomarker active only inside the
  feature_013-positive stratum than a treatment.

----------------------------------------------------------------------
4. SUBGROUP CHARACTERIZATION OF feature_023 (Iterations 13–16, 18–21)
----------------------------------------------------------------------

Stratified analyses make the picture concrete:

  feature_023 effect by feature_013:
      feature_013 = 0 (n=32,572) :  −0.020 mo  (p = 0.41)  — null
      feature_013 = 1 (n=17,428) :  +1.072 mo  (p = 5e-134) — strong

  feature_023 effect by feature_028:
      feature_028 = 0 (n=32,635) :  +0.554 mo  (p = 7e-81)  — strong
      feature_028 = 1 (n=17,365) :  −0.003 mo  (p = 0.93)   — null

  feature_023 effect by feature_018:
      feature_018 = 0 (n=40,940) :  +0.436 mo  (p = 5e-67)  — strong
      feature_018 = 1 (n= 9,060) :  +0.018 mo  (p = 0.69)   — null

  feature_023 effect by feature_033 quartile:
      Q1 ( 1.0– 7.5)  +0.612 (p = 4e-37)
      Q2 ( 7.6–12.2)  +0.539 (p = 1e-29)
      Q3 (12.3–19.6)  +0.238 (p = 2e-08)
      Q4 (19.7–100)   +0.042 (p = 0.28) — null

These four modifiers act roughly multiplicatively. A formal four-way OLS
with all four interaction terms in the model confirms each is independently
significant (all p < 1e-12) with sign pattern matching the stratifications.

After full multivariable adjustment within feature_013==0, the feature_023
coefficient is −0.004 (p = 0.18) — confirming that all of the apparent
effect is in feature_013==1.

----------------------------------------------------------------------
5. JOINT SUBGROUP DEFINITION AND REFINEMENT (Iterations 18–19, 22–23)
----------------------------------------------------------------------

Stacking the binary modifiers:

  Subgroup                                        n     Δ pfs (mo)     p
  ---------------------------------------------   ----  ----------     -----
  All                                             50000   +0.36       6e-59
  feature_013 = 1                                 17428   +1.07       5e-134
  feature_013 = 1 & feature_028 = 0               11405   +1.62       1e-173
  feature_013 = 1 & feature_028 = 0 & feature_018 = 0
                                                   9326   +1.97       4e-199
  Add feature_033 < median (~12.2):
  feature_013 = 1 & feature_028 = 0 & feature_018 = 0
   & feature_033 < median                          4588   +3.27       2e-241

Outside the four-predicate subgroup, the feature_023 effect is +0.05 mo
(p = 0.015 by sample-size alone — clinically null).

Within the favorable f013=1 & f028=0 & f018=0 cell, feature_016 status
does NOT further modify the feature_023 effect (treatment benefit is
~+2.05 vs +1.87 mo in the two feature_016 strata; feature_016 is a
prognostic, not a treatment-effect modifier).

A balance check confirmed that feature_023 is NOT randomly assigned —
feature_029 in particular differs strongly between feature_023=1/0
groups (p ≈ 0). The reported associations therefore mix true heterogeneous
treatment effect with residual confounding; the direction and modifier
pattern is robust under full adjustment, but the magnitudes should be
read as subgroup-specific associations, not pure causal estimates.

An exhaustive search over all pairs of binary features (with cell n ≥ 1500)
identified no two-feature definition giving a larger feature_023 mean
PFS gain than (feature_013==1 & feature_028==0); the (feature_013==1 &
feature_018==0) cell was a close second. Refining further required the
continuous biomarker feature_033.

----------------------------------------------------------------------
6. SUPPORTED VS. REFUTED HYPOTHESES
----------------------------------------------------------------------

Supported with strong evidence (p < 1e-50):
  - feature_012 positively correlated with pfs_months (h2)
  - feature_005 level 0 > 1 > 2 in pfs_months (h3, h6)
  - feature_013==1, feature_029==1 better pfs_months (h8, h9)
  - feature_006==1, feature_001==1, feature_028==1, feature_018==1 worse
    pfs_months (h10–h13)
  - feature_024 positive, feature_033 negative, feature_004 negative
    associations with pfs_months (h14–h16)
  - feature_023 effect on pfs_months is highly heterogeneous, modified
    by feature_013, feature_028, feature_018, feature_033 (h21, h30)
  - feature_023 benefit confined to feature_013==1 (h22)
  - feature_028==1 abolishes feature_023 benefit (h23)
  - feature_018==1 abolishes feature_023 benefit (h24)
  - feature_023 benefit declines monotonically with feature_033 quartile
    and is null in Q4 (h25)
  - Within the joint favorable subgroup, feature_023 yields ~+3 mo PFS
    gain (h27, h28, h34)
  - feature_023 yields no benefit in feature_013==0, even after
    full multivariable adjustment (h29)
  - Treatment allocation is not random — feature_023 is correlated
    with feature_029 and several biomarkers (h31)

Refuted / null:
  - No clinically meaningful treatment-effect heterogeneity for
    feature_030, feature_019, or feature_011 (h18–h20)
  - feature_016 does not further modify feature_023 effect within the
    favorable subgroup (h32)
  - feature_005 (stage-like) does not modify feature_023 effect — the
    magnitude of treatment benefit (~+1 mo at f013=1) is similar at
    every stage level

----------------------------------------------------------------------
7. FINAL CONSOLIDATED SUBGROUP HYPOTHESIS
----------------------------------------------------------------------

TREATMENT  : feature_023 (binary, prevalence 0.64)
OUTCOME    : pfs_months (continuous, mean 4.69 in cohort)
DIRECTION  : positive (treatment prolongs pfs_months)

SUBGROUP DEFINING BENEFIT (all conditions required):
    feature_013 == 1
    feature_028 == 0
    feature_018 == 0
    feature_033 < median (≈ 12.2)

  • Inside subgroup (n = 4,588):
        feature_023 = 1: mean PFS 8.95 mo
        feature_023 = 0: mean PFS 5.67 mo
        Δ = +3.27 mo (p = 2 × 10⁻²⁴¹)

  • Outside subgroup (n = 45,412):
        Δ = +0.05 mo (p = 0.015, clinically null)

UNFAVORABLE VALUES THAT SUPPRESS THE TREATMENT EFFECT
(any one of these moves a patient out of the benefit subgroup):
    feature_013 == 0   (no benefit at all)
    feature_028 == 1   (cancels benefit)
    feature_018 == 1   (cancels benefit)
    feature_033 in top quartile (≥ ~20)  (cancels benefit)

Interpretation: the data structure is consistent with a biomarker-defined
treatment, where feature_013 is the eligibility/positive-predictive
biomarker, feature_028 and feature_018 mark adverse states that abolish
benefit, and feature_033 is a continuous tumor-burden / disease-activity
modifier whose rising value progressively erodes the treatment effect.
The pattern is stable across feature_005 (stage) levels, meaning the
benefit subgroup is biomarker-defined rather than stage-defined.

The remaining candidate "treatment" indicators (feature_030, feature_019,
feature_011) showed no meaningful heterogeneity and no main effect; if
they are real treatments in the underlying study, the data are consistent
with them being uniformly inert with respect to pfs_months in this cohort.
