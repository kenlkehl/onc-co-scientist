Oncology Dataset Analysis Summary
Dataset: ds001_aml  |  N = 50,000 patients  |  Outcome: objective_response (binary)
Cohort base rate: 18.75% responders.
=================================================================================

1. Univariate screening (Iteration 1)
-------------------------------------
A point-biserial / chi-square scan of all 35 anonymized features against
objective_response identified 10 variables with p < 1e-5:

  Negative associations with response:
    - feature_004 (ordinal 0/1/2)   p = 1.8e-90  (response 23%/18%/12%)
    - feature_020 (continuous 0-23) p = 8.0e-31
    - feature_024 (continuous 20-100) p = 2.3e-16
    - feature_026 (binary)          p = 3.3e-6
    - feature_027 (continuous 0-300, skewed) p = 6.0e-6
    - feature_009 (binary)          p = 3.5e-5
    - feature_022 (continuous 0.5-500) p = 1.8e-4

  Positive associations with response:
    - feature_014 (binary)  +6.7 pp  p = 5.8e-70
    - feature_001 (binary)  +5.6 pp  p = 1.9e-55
    - feature_021 (binary)  +5.4 pp  p = 6.6e-51
    - feature_029 (continuous 1.7-5.5) p = 2.8e-7

A multivariable logistic regression of objective_response on all 35 features
(Iteration 5) confirmed the same set of independent predictors and added no
new ones; in particular, feature_011 (which has the value range and mean of an
age-in-years variable, 18-92, mean ~68) had no detectable main effect
(adjusted p = 0.67), and the two clinically-recognisable laboratory ranges
(feature_023 ~ sodium 128-152, feature_002 ~ potassium 2.5-6.3,
feature_017 ~ calcium 7.3-11.8) were all p > 0.7 in the adjusted model.

2. Three-way AND-interaction discovery (Iterations 6-10)
--------------------------------------------------------
The three positively-associated binary features (feature_001, feature_014,
feature_021) were tested for joint behaviour. Their pairwise cell sizes match
statistical independence to within 1%, so the marginal main effects are not
explained by overlap.

The 8-cell joint table of (feature_001, feature_014, feature_021) revealed a
striking pattern:

   f001  f014  f021     n     response
   0     0     0    12,533    17.2%
   0     0     1     8,383    16.2%
   0     1     0     5,404    16.0%
   0     1     1     3,624    15.7%
   1     0     0     8,458    16.4%
   1     0     1     5,612    16.9%
   1     1     0     3,585    16.0%
   1     1     1     2,401    63.3%   <-- triple-positive only

Seven of the eight cells lie in 16-17%. Only the all-three-positive cell
jumps to 63.3% -- a 47-percentage-point excess that cannot be explained by
additive main effects.

Formal logistic regression with all main effects, two-way and three-way
product terms (Iteration 8) showed that the three-way interaction term
(feature_001:feature_014:feature_021) had coefficient +2.13 (OR ~ 8.4,
p < 1e-90, z = 20.4); main effects and two-way terms were all near zero
or slightly negative. The "count of positives among the three" was flat at
16-17% for counts 0, 1, 2 and jumped only at count 3 -- the relationship is
a pure logical AND, not a graded dose-response. Stratified analysis confirmed
symmetry: the marginal effect of any one of the three features is essentially
zero in every (other-two) stratum except (other-two = 1,1), where it is +47-48
percentage points. All three are functionally interchangeable and each is
strictly necessary.

Interpretation: This is the signature of either a triple-combination treatment
or a single intervention whose effect requires the simultaneous presence of
two specific biomarkers. The data alone cannot distinguish which of the three
indicators is the modifiable treatment vs. which are biomarkers.

3. Suppressors of the triple-positive effect (Iterations 11-13)
---------------------------------------------------------------
Within the triple-positive subgroup (n = 2,401, baseline 63.3% response), a
chi-square scan of all remaining features identified two binary suppressors:

   feature_026 = 0 (n=2,025) -> 71.7%   vs   feature_026 = 1 (n=376) -> 18.4%
   feature_009 = 0 (n=2,125) -> 69.1%   vs   feature_009 = 1 (n=276) -> 18.8%

When BOTH suppressors are absent, response is 78.6% (n=1,790); when either is
present, response collapses back to ~18% -- essentially the cohort baseline.

Formal interaction tests confirmed: triple x feature_026 = -2.44 (p = 2e-62)
and triple x feature_009 = -2.26 (p = 4e-42); both interactions almost
exactly cancel the triple-positive log-odds gain. By contrast, the triple x
feature_004 interaction was non-significant (p = 0.34): feature_004 acts as a
constant prognostic main effect (about -0.39 per ordinal level) both inside
and outside the triple-positive subgroup.

4. Refined subgroup definition (Iterations 15-17)
-------------------------------------------------
The refined high-response subgroup is:

   feature_001 = 1  AND  feature_014 = 1  AND  feature_021 = 1
                AND  feature_026 = 0  AND  feature_009 = 0

   n = 1,790  (3.58% of cohort)
   Response = 78.6%  vs  16.5% in the complement
   Risk difference = +62.1 percentage points  (chi-square p << 1e-300)
   Odds ratio ~ 19

Adding feature_004 = 0 to the conjunction yields the strictest subgroup
(n = 617, response 85.9%, risk difference +68 pp). The full feature_004
gradient within the refined subgroup is 85.9% / 77.2% / 66.8% across
levels 0 / 1 / 2 (p = 4e-10).

5. Heterogeneity searches that came up negative (Iterations 18-21)
------------------------------------------------------------------
- feature_011 (age, 18-92): the triple-positive risk difference is
  +48.8 / +47.0 / +45.3 / +45.8 pp across age groups <60 / 60-69 / 70-79 / >=80.
  Logistic interaction triple x age = -0.0036 per year (p = 0.38).
  No age modification.
- All other binary features (feature_005, _010, _015, _016, _018, _019, _025,
  _006, _008, _030, _032): no significant triple x feature interaction
  (smallest p ~ 0.12) and none meaningfully shifts response within the
  refined subgroup.
- All other continuous features (feature_022, _024, _027, _028, _029, _034,
  _007, _012, _031, _013, _033, _003, _035, _023, _002, _017): no significant
  interaction with the triple indicator. Within the refined subgroup, only
  feature_020 reached r = -0.084 (p = 4e-4); it appears to be a weak,
  uniformly-acting prognostic rather than a true effect modifier.
- Exhaustive pairwise AND-screen across the 16 binary features: no alternative
  high-response combination outside the (feature_001, feature_014, feature_021)
  family reaches even 35% response.
- Among the 47,599 non-triple-positive patients, no subgroup reaches >25%
  response in any cell of >100 patients; the best stratum is feature_004 = 0
  (response 20.8%, n = 16,762).

6. Adjusted-model confirmation (Iteration 22)
---------------------------------------------
A logistic model containing the triple indicator, feature_026, feature_009,
the two relevant interactions, the ordinal feature_004, and the strongest
continuous prognostics (feature_020, feature_024, feature_022, feature_029,
feature_027) plus age (feature_011) yields:

   triple                +2.98   (p << 1e-100)
   triple x feature_026  -2.75   (p =  8e-74)
   triple x feature_009  -2.62   (p =  4e-52)
   feature_004           -0.39   (p << 1e-90)
   feature_020           -0.040  (p =  8e-35)
   feature_024           -0.0056 (p =  1e-18)
   feature_022           -0.0021 (p =  3e-5 )
   feature_029           +0.130  (p =  7e-8 )
   feature_027           -0.0074 (p =  7e-7 )
   feature_011 (age)     -0.0002 (p =  0.85 )

The triple x suppressor interactions remain by far the dominant terms after
adjustment for every continuous prognostic.

7. Final treatment-effect subgroup hypothesis (Iteration 25)
------------------------------------------------------------
   Subgroup:   feature_001 = 1  AND  feature_014 = 1  AND  feature_021 = 1
               AND  feature_026 = 0  AND  feature_009 = 0
   Outcome:    objective_response
   Direction:  POSITIVE -- response in subgroup 78.6% vs 16.5% outside.
   Magnitude:  Absolute risk difference +62 percentage points; OR ~ 19.

Necessary conditions discovered:
  - All three of {feature_001, feature_014, feature_021} must be present
    (none alone, and no pair, is sufficient -- response is flat at 16-17%
     for 0, 1, or 2 of them).
  - Both feature_026 and feature_009 must be ABSENT -- either, set to 1,
    completely abolishes the subgroup benefit.

Further graded by:
  - feature_004 (ordinal 0/1/2): response 85.9% / 77.2% / 66.8% within the
    subgroup. feature_004 acts as a constant prognostic, not as a moderator
    of the triple effect.

Variables that do NOT modify the treatment effect:
  - feature_011 (age 18-92), feature_010, feature_005, all other binary
    features, and all other continuous features tested.

This is the best-supported treatment-by-biomarker subgroup hypothesis in
the cohort. Whether the modifiable "treatment" is feature_001, feature_014,
or feature_021 cannot be determined from the data: the three are
exchangeable in their effect, and any one of them can be cast as the
treatment with the other two interpreted as enabling biomarkers (and
feature_026, feature_009 as resistance markers).
