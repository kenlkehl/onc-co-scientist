Analysis summary — ds001_crc (50,000 patients; outcome = pfs_months)
=====================================================================

Across 20 iterations I probed treatment main effects, biomarker main effects,
canonical CRC treatment×biomarker interactions, prognostic clinical/lab
features, sociodemographics, SNPs, multivariable adjustment, and three-way
clinical phenotypes. The cohort is large enough (n=50,000) that even modest
effects reach statistical significance, so I focus on direction, magnitude,
and which signals survive multivariable adjustment (final R² ≈ 0.84).

==========================================================
1. Treatment main effects (Iteration 1)
==========================================================
Of the six treatments, only one shows a strong unadjusted main effect on PFS:

  - treatment_regorafenib   diff = +0.97 months (p < 1e-216, n=10,022)  ✅
  - treatment_cetuximab     diff = -0.04 months (p = 0.088)             —
  - treatment_bevacizumab   diff = -0.02 months (p = 0.35)              —
  - treatment_pembrolizumab diff = +0.005 months (p = 0.86)             —
  - treatment_encorafenib   diff = +0.005 months (p = 0.87)             —
  - treatment_trastuzumab+tucatinib diff = -0.04 months (p = 0.24)      —

Regorafenib is the only treatment associated with longer PFS at both
unadjusted and multivariable levels. The other five are essentially
indistinguishable from no exposure in marginal analysis.

==========================================================
2. Biomarker main effects (Iteration 2)
==========================================================
  - kras_mutation     diff = -0.33 months (p ≈ 2e-58)   supported (worse)
  - braf_v600e        diff = -0.23 months (p ≈ 3e-7)    supported (worse)
  - nras_mutation     diff = +0.22 months (p = 7e-4)    direction REVERSED
                                                         from prior expectation
  - msi_high          diff = -0.02 months (p = 0.68)    null
  - her2_amplified    diff = -0.06 months (p = 0.36)    null
  - tp53_mutation     diff = -0.02 months (p = 0.38)    null
  - pik3ca_mutation   diff = -0.02 months (p = 0.53)    null
  - ntrk_fusion       diff = +0.04 months (p = 0.79)    null

KRAS and BRAF V600E behave like classic adverse prognostic markers.
NRAS surprisingly trends with longer PFS — modest magnitude but with n=1,506.
MSI-high shows no PFS difference, unusual for CRC (where MSI-H normally
carries a better prognosis at metastatic and earlier stages).

==========================================================
3. Prognostic baseline (Iteration 3)
==========================================================
All seven proposed prognostic relationships were significant; one direction
was OPPOSITE the canonical clinical expectation:

  - stage_iv:           diff = -1.35 months (p ≈ 0)        ✅ as expected
  - right_sided_primary diff = -0.31 months (p ≈ 1e-49)    ✅ as expected
  - ecog_ps             slope = -1.19 months/unit (p ≈ 0)  ✅ as expected
  - albumin_g_dl        slope = +0.47 months/(g/dL)        ✅ as expected
  - ldh_u_l             slope = -0.0003 (p = 0.013)        ✅ as expected
  - cea_ng_ml           slope = -0.005 (p = 3e-17)         ✅ as expected
  - age_years           slope = +0.18 months/year (p ≈ 0)  REVERSED — older
                                                            age associates with
                                                            longer PFS in this
                                                            cohort.

The positive age slope is striking — it persists in multivariable models
(coef +0.176) and is unlikely to be confounded by the variables we adjusted
for. It may reflect treatment-selection / lead-time bias in this EHR cohort,
slower-growing tumors in older patients, or a dataset-specific pattern.

==========================================================
4. Canonical CRC treatment × biomarker interactions
   (Iterations 4–7) — ALL NULL
==========================================================
This dataset does NOT replicate the textbook precision-oncology pairings:

  - cetuximab × KRAS-mutant interaction:   coef = +0.020 (p = 0.66)  null
  - cetuximab × NRAS-mutant interaction:   coef = -0.007 (p = 0.96)  null
  - cetuximab × RAS-WT interaction:        coef = -0.017 (p = 0.71)  null
  - encorafenib × BRAF V600E interaction:  coef = -0.14  (p = 0.38)  null
                (within BRAF V600E, n=2,272: enc diff = -0.13, p=0.33)
  - pembrolizumab × MSI-high interaction:  coef = +0.002 (p = 0.99)  null
                (within MSI-H, n=2,513: pem diff = +0.007, p=0.96)
  - trast+tuc × HER2-amplified interaction: coef = +0.06 (p = 0.78)  null
                (within HER2+, n=1,504: trt diff = +0.02, p=0.93)

Within each biomarker-positive subgroup the corresponding "matched" therapy
shows essentially zero effect. Because subgroup n's are 1,500–2,500 and the
overall PFS SD is ≈2.3, we have power to detect effects of ~0.2 months in
the smaller subgroups; therefore the absence of large benefit is informative,
not just underpowered. **In this dataset, no canonical CRC genotype-targeted
therapy benefit is detectable.**

==========================================================
5. Sidedness × cetuximab (Iteration 8) and three-way RAS-WT × side × cet
   (Iteration 16)
==========================================================
  - cetuximab × right_sided interaction: coef = +0.026 (p = 0.58)  null
  - In RAS-WT, left-sided (n=18,049): cet_diff = -0.06 months (p = 0.13)
  - In RAS-WT, right-sided (n=8,948): cet_diff = -0.04 months (p = 0.44)

The composite "RAS-WT + left-sided + cet" group does have longer PFS than
the rest (+0.30 months, p ≈ 1.6e-16), but that signal is driven by the
prognostic pieces (RAS-WT + left-sided), not by cetuximab — within RAS-WT
left-sided patients cetuximab confers no measurable benefit.

7. Mets, symptoms, demographics, SNPs (Iterations 10–13)
==========================================================
  - liver_mets, bone_mets, adrenal_mets, mets_count: ALL null on PFS.
    (Surprising — these usually carry adverse prognostic weight.)
  - All five symptom grades (fatigue, pain, dyspnea, appetite_loss) and
    the composite symptom-burden score: NULL (all p > 0.1).
  - Sex, rural residence, smoking pack-years, education years: NULL.
  - race_ethnicity ANOVA F = 0.87 (p = 0.48) — null.
  - insurance_type ANOVA F = 1.48 (p = 0.22) — null.
  - 27 SNPs screened; 3 reach univariate p<0.05 (rs1050828 +0.09 p=0.017,
    rs1801131 -0.05 p=0.027, rs1801197 -0.05 p=0.046). Number of "hits" is
    consistent with chance under 27 tests at α=0.05 (≈1.4 expected, 3 seen,
    none survive Bonferroni). No reliable SNP signal.

==========================================================
10. Synthesis and conclusions
==========================================================
**Hypotheses supported.**
   - All seven prognostic-feature hypotheses (ECOG, stage, albumin, LDH, CEA,
     sidedness, weight loss). Note the age slope is statistically supported
     but its direction is opposite the textbook expectation.
   - KRAS, BRAF V600E adverse-prognostic main effects (h2.1, h2.3).
   - Regorafenib main-effect benefit (h1.6) — large, robust to multivariable
     adjustment, and uniform across lines of therapy.
   - Prognostic strength of the "good-prognosis" composite RAS-WT + left-
     sided (h16.1) — but cetuximab adds no benefit on top of those features.

**Hypotheses refuted (or not supported).**
   - Cetuximab × KRAS / NRAS / RAS-WT interaction (h4.1–4.3).
   - Encorafenib × BRAF V600E interaction (h5.1–5.2).
   - Pembrolizumab × MSI-high interaction (h6.1–6.2).
   - Trastuzumab+Tucatinib × HER2-amplified interaction (h7.1–7.2).
   - Cetuximab × sidedness, cetuximab benefit within RAS-WT left-sided
     (h8.1–8.2, h16.2).
   - Bevacizumab modifier hypotheses (h17.1–17.3).
   - Regorafenib × line of therapy (h18.1).
   - Pembrolizumab × NLR (h19.1).
   - All sociodemographic predictors and the SNP global enrichment (h12.1–
     12.6, h13.global).
   - Symptom-burden hypotheses (h11.1–11.5).
   - Most metastatic-site main effects (h10.1–10.4).
   - MSI-high better prognosis hypothesis (h2.4).

**Bottom line.** In this 50,000-patient CRC cohort, PFS is driven mostly by
classical host-condition prognostic features (ECOG, stage, albumin, weight
loss, LDH, CEA, age, sidedness) plus negative effects of KRAS and BRAF V600E
mutations and a strong positive effect of regorafenib. The textbook CRC
biomarker-targeted-therapy interactions (cetuximab in RAS-WT, encorafenib in
BRAF V600E, pembrolizumab in MSI-H, trastuzumab+tucatinib in HER2+) are NOT
detectable in this dataset despite ample power, and several "expected"
adverse-prognostic features (liver/bone/adrenal mets, MSI-high direction,
NLR, symptom burden) are also absent or reversed — suggesting either a
dataset where treatment assignment is largely uninformative for PFS or
where the simulated/aggregated treatment-effect structure deliberately
omits these canonical interactions.
