Analysis summary - ds001_aml cohort (N = 50,000)
===================================================

Outcome: objective_response (binary). Overall response rate: 16.9% (8,452/50,000).

This summary walks through 25 iterations of hypothesis-driven analyses, organized
by topic (demographics, disease biology, treatment-mutation interactions, labs,
comorbidities, pharmacogenomics, sociodemographic factors, and multivariable
models). Each finding reports a signed effect estimate (on the response-rate or
log-odds scale, as appropriate) and a p-value. Significance threshold: p < 0.05.


1. Demographics and performance status (iter 1)
-----------------------------------------------
- age_years: NOT associated with response (univariable log-OR per year = +0.0004,
  p = 0.72). Refuted as a univariable predictor.
- sex_female: NOT associated (diff = +0.003, p = 0.32). Refuted.
- ecog_ps: STRONGLY negatively associated with response — log-OR per ECOG point
  = -0.384 (p = 1.8e-97). Each additional ECOG point lowers the odds of
  response by ~32%. SUPPORTED.

2. Disease-biology main effects (iter 2-3)
------------------------------------------
- secondary_aml: NOT associated (diff = -0.0005, p = 0.90). Refuted.
- complex_karyotype: no main-effect difference in response (diff = -0.001,
  p = 0.81). Refuted as a univariable predictor (but see iter 7 for treatment
  interaction).
- unfit_for_intensive: no main-effect difference (diff = +0.0007, p = 0.83).
  Refuted univariably.
- blast_pct_marrow: higher marrow blasts -> lower response, log-OR = -0.0057
  per percentage point (p = 1.2e-19). SUPPORTED.
- flt3_itd, flt3_tkd, idh2_mutation, npm1_mutation, tp53_mutation: NO main-
  effect differences in response rate (all p > 0.10). All refuted as main
  effects (but most participate in treatment-mutation interactions, see below).
- idh1_mutation: HIGHER response in IDH1-mutated patients (22.1% vs 16.5%,
  diff = +5.6 percentage points, p ~ 0). STRONGLY SUPPORTED. This was the
  largest mutation main effect in the cohort.

3. Treatment x targetable-mutation interactions (iter 4-7)
----------------------------------------------------------
- treatment_midostaurin x flt3_itd: NO benefit; in fact, the response rate among
  FLT3-ITD+ patients on midostaurin was 16.1% vs 17.4% off, tx-effect = -0.012,
  p = 0.22. Refuted (no on-target benefit detected).
- treatment_gilteritinib x flt3_itd: NO benefit; tx-effect within FLT3-ITD+
  = -0.003, p = 0.69. Refuted.
- treatment_midostaurin x flt3_tkd: NS (p = 0.87). Refuted.
- treatment_gilteritinib x flt3_tkd: numerically lower response (14.1%) but NS
  (p = 0.33). Refuted.
- treatment_ivosidenib x idh1_mutation: SIGNIFICANT but NEGATIVE — within IDH1+,
  ivosidenib-treated patients responded LESS (17.4%) than ivosidenib-untreated
  IDH1+ patients (22.5%), tx-effect = -0.051, interaction p = 0.027. The
  observed IDH1 main-effect benefit therefore appears concentrated in IDH1+
  patients NOT receiving ivosidenib (refuted as expected/positive interaction;
  significant in opposite direction; revisited in iter 21).
- treatment_enasidenib x idh2_mutation: tx-effect within IDH2+ = -0.023, NS
  (p = 0.45). Refuted (no on-target enasidenib benefit detected).
- treatment_venetoclax_azacitidine x npm1_mutation: SUPPORTED. Within NPM1+,
  ven/aza-treated patients responded 19.3% vs 16.0% in NPM1+ patients NOT on
  ven/aza, tx-effect = +0.034 (p = 6.6e-7); the interaction log-OR was +0.26
  (p = 6.6e-7). This is the strongest treatment-mutation signal in the dataset.
- treatment_venetoclax_azacitidine x any IDH mutation: SUPPORTED. Within any-IDH
  mutated patients, ven/aza-treated responded 21.9% vs 17.0% off, tx-effect
  = +0.050 (interaction p = 1.9e-7). However, this is largely driven by IDH1
  carriers in the off-treatment arm and the NPM1-co-mutated subset; see notes.
- treatment_venetoclax_azacitidine main effect: a small but significant overall
  benefit (17.3% vs 16.6%, diff = +0.007, p = 0.031). SUPPORTED.
- treatment_venetoclax_azacitidine x tp53_mutation: NS (p = 0.29). Refuted as
  TP53-modifying.
- treatment_7plus3 x complex_karyotype: SIGNIFICANT NEGATIVE interaction —
  within complex-karyotype+ patients, 7+3-treated responded 15.5% vs 17.9% off,
  tx-effect = -0.023, interaction p = 0.0016. SUPPORTED in the negative
  direction: 7+3 appears harmful in patients with complex karyotype.
- treatment_7plus3 x unfit_for_intensive: NS (p = 0.50).
- treatment_7plus3 x age_years: NS interaction (p = 0.61). The 7+3 age effect
  is not detectable.
- treatment_venetoclax_azacitidine x unfit_for_intensive: SUPPORTED. Within unfit
  patients, ven/aza-treated responded 17.9% vs 16.3% off, tx-effect = +0.016,
  interaction p = 0.036.

4. Continuous lab predictors (iter 8-10)
----------------------------------------
- wbc_k_per_ul: higher WBC -> lower response, log-OR per unit = -0.0022,
  p = 1.9e-5. SUPPORTED.
- albumin_g_dl: higher albumin -> higher response, log-OR per g/dL = +0.115,
  p = 1.4e-6. SUPPORTED.
- ldh_u_l: NS (p = 0.35). Refuted as a univariable predictor (somewhat
  surprising clinically; the synthetic generator did not encode an LDH effect).
- hemoglobin_g_dl, platelets_k_ul, anc_k_ul: all NS univariably. Refuted.
- crp_mg_l: higher CRP -> lower response, log-OR = -0.007, p = 2.8e-6. SUPPORTED.
- nlr: NS (p = 0.56).
- weight_loss_pct_6mo: STRONGLY negatively associated, log-OR = -0.040 per
  percentage-point, p = 7.4e-35. SUPPORTED — among the most consistent signals.
- fatigue_grade, pain_nrs: NS univariably.
- appetite_loss_grade: weakly significant negative effect (p = 0.038). SUPPORTED.
- total_bilirubin_mg_dl, creatinine_mg_dl, ast, alt, inr: all NS.
- alkaline_phosphatase_u_l: marginally significant POSITIVE association
  (log-OR = +0.00047, p = 0.048) — likely chance given direction is biologically
  unexpected and effect size is tiny. Borderline.

5. Comorbidities and prior therapy (iter 11-12)
-----------------------------------------------
- diabetes, hypertension, CKD, heart failure, COPD, prior malignancy: all NS
  univariably (all p > 0.12). Refuted as predictors of objective response in
  this cohort.
- prior_chemotherapy, prior_radiation, prior_targeted_therapy: NS.
- prior_lines_of_therapy: NS (p = 0.52).
- years_since_diagnosis: borderline negative (p = 0.056). Inconclusive.

6. Composite molecular risk groups (iter 13)
--------------------------------------------
- Favorable class (npm1_mutation=1 AND flt3_itd=0): no marginal main-effect
  difference (diff = +0.004, p = 0.29). Refuted as a stand-alone marker.
- High-risk class (tp53_mutation=1 AND complex_karyotype=1): NS (n was small;
  diff = -0.009, p = 0.51). Refuted univariably.
- REFINED hypothesis - venaza benefit concentrated in fav_npm1: SUPPORTED. In
  the favorable NPM1+/FLT3-ITD- subset, ven/aza-treated responded 19.1% vs 16.0%
  off, tx-effect = +0.031; in non-favorable patients, ven/aza had essentially no
  effect (16.8% vs 16.8%). Interaction log-OR = +0.214, p = 1.4e-4. This
  reinforces the iter-6 NPM1 finding and shows that the ven/aza benefit is
  concentrated in (and possibly driven by) the favorable NPM1-mutant subgroup.

7. Pharmacogenomic SNPs (iter 14-16)
-------------------------------------
Across the 24 SNP carrier-status main-effect tests, only two reached p<0.05,
consistent with a multiple-testing chance rate:
- snp_rs2032582 (ABCB1): carriers had slightly LOWER response (16.5% vs 17.2%,
  diff = -0.007, p = 0.049). Borderline.
- snp_rs1050828 (G6PD): carriers had HIGHER response (18.5% vs 16.8%,
  diff = +0.017, p = 0.005). SUPPORTED (with caveat that 24 tests were run).
- All other SNPs (CYP2C19, CYP2C9, CYP2D6, NQO1, MTHFR, APOE, BCHE, MDR1,
  COMT, PON1, etc. carrier flags): NS. Refuted.

SNP x treatment interactions (iter 24):
- snp_rs429358 (APOE) x treatment_7plus3: SIGNIFICANT positive interaction
  log-OR = +0.160, p = 0.017. Among APOE-rs429358 carriers, 7+3 appears more
  beneficial; modest signal that warrants validation given multiple testing.
- snp_rs4244285 (CYP2C19*2) x ven/aza: NS.
- snp_rs1801133 (MTHFR C677T) x ven/aza: NS.

8. Sociodemographic factors (iter 17)
-------------------------------------
- race_ethnicity (omnibus): SIGNIFICANT (chi-square p = 0.010). White
  patients had the highest response (17.3%) and "other" the lowest (14.7%);
  black patients 15.7%, Asian 16.3%, Hispanic 16.7%. The absolute spread is
  ~2.6 percentage points; the omnibus test is significant given the sample
  size. SUPPORTED, but small in magnitude and confounded with other features
  (insurance, comorbidities).
- insurance_type (omnibus): NS (p > 0.05).
- rural_residence: NS (diff = -0.005 to +0.002, p > 0.10). Refuted.
- education_years: NS univariably.
- smoking_pack_years: NS univariably.

9. Vital signs and non-AML disease markers (iter 18-19)
-------------------------------------------------------
- bmi, systolic_bp_mmhg, heart_rate_bpm, spo2_pct, dyspnea_grade: all NS
  univariably. Refuted.
- Negative-control markers (ca_125, cea, psa, tsh, her2_amplification,
  braf_v600e): all NS. Refuted, as expected for an AML cohort — these
  variables behave like noise for this outcome.

10. Refined IDH-inhibitor analyses (iter 20-21)
-----------------------------------------------
- treatment_enasidenib x any_idh: tx-effect in any-IDH+ = -0.001, NS
  (interaction p = 0.45). Refuted.
- treatment_ivosidenib x any_idh: tx-effect = -0.029, NS (interaction p = 0.20).
  Refuted as an "IDH-inhibitor benefit" hypothesis.
- any_idh_inhib x any_idh: tx-effect in IDH+ = -0.014, NS. Refuted.
- REFINED: ivosidenib in idh1_mutation+ (h21_ivo_idh1_refined): the IDH1+
  benefit is concentrated in IDH1+ patients NOT given ivosidenib (response
  22.5%); IDH1+ on ivosidenib responded only 17.4%. Interaction p = 0.027.
  Supported in negative direction.
- The IDH1-mutation effect is therefore best characterized as an
  unconditional / treatment-independent biomarker of higher response, NOT as
  a marker of ivosidenib benefit in this cohort.

11. Multivariable models (iter 22, 25)
--------------------------------------
Multivariable logistic regression adjusting for age, sex, ECOG, secondary AML,
unfit-status, complex karyotype, all major mutations, key labs (WBC,
blast_pct_marrow, albumin, LDH), and the six treatment indicators:
- idh1_mutation: adjusted log-OR = +0.360 (p = 2.2e-17). Independently
  associated with higher response after extensive adjustment. SUPPORTED.
- complex_karyotype: NS in the multivariable model (refuted as a main
  effect; effect captured by interaction with 7+3, see iter 7).
- unfit_for_intensive: NS in the multivariable model after adjusting for
  ECOG and labs. Refuted (its apparent prognostic role is mediated by ECOG).
- ecog_ps, blast_pct_marrow, albumin, weight_loss_pct_6mo: independent
  prognostic factors (all p < 0.001 in the multivariable model).
- age_years: NS independently.
Final model with treatment-mutation interactions confirmed:
- treatment_venetoclax_azacitidine:npm1_mutation interaction term:
  adjusted log-OR = +0.261 (p = 8.0e-7). SUPPORTED. Independent of all
  other adjustments, NPM1+ patients derive a substantially larger response
  benefit from venetoclax/azacitidine than NPM1-wt patients.
- idh1_mutation main term in interaction model: adjusted log-OR = +0.387
  (p = 1.5e-18). SUPPORTED.
- unfit_for_intensive: NS in the adjusted model. Refuted independently.
- fav_npm1 (NPM1+/FLT3-ITD-): NS as a main-effect predictor in the adjusted
  model that already includes NPM1 and FLT3-ITD separately (composite did
  not add independent information).

12. Subgroup-specific treatment analyses (iter 23)
--------------------------------------------------
- Older (age >= 70) AND unfit: ven/aza response = 18.0% vs 16.4% off (n = 5,968
  in subset), diff = +0.016, p = 0.10 in subgroup. Trend in expected direction
  but not significant in this slice (consistent with a real but modest benefit).
- Younger (<60) AND fit subset: 7+3 response = 16.6% vs 16.6%, NS. No effect
  detected.
- treatment_venetoclax_azacitidine x age_years interaction: interaction log-OR
  per year = -3.9e-5, p = 0.99. NO age modification of the ven/aza effect
  detected. Refuted.


Overall conclusions
===================

Outcome base rate: ~17%. Despite the very large sample (N = 50,000), main
effects of most clinical features are small or absent — consistent with the
dataset being a synthetic / simulated cohort where most features are
intentionally near-null and a few prognostic and predictive signals are encoded.

Robust findings (supported across analyses, large effect sizes or very small
p-values):
1. ECOG performance status is the dominant prognostic factor: each ECOG
   point reduces the odds of response by ~32% (log-OR = -0.38).
2. Higher baseline marrow blast percentage is independently negatively
   associated with response (log-OR per % = -0.006).
3. Higher baseline albumin (log-OR per g/dL = +0.115) and lower CRP
   (log-OR per mg/L = -0.007) are independently associated with higher
   response — a "fitness/inflammation" axis.
4. Greater 6-month weight loss is one of the strongest negative prognostic
   features (log-OR = -0.040; p ~ 1e-34).
5. Higher WBC at baseline is modestly negatively associated with response.
6. IDH1 mutation is the strongest molecular biomarker of response in this
   cohort (response 22% vs 16% wild type; adjusted log-OR ~ +0.39). This
   association is treatment-independent — IDH1+ patients respond more whether
   or not they receive ivosidenib.
7. Treatment-biomarker interaction: venetoclax/azacitidine is more
   effective in NPM1-mutated patients than in NPM1-wild-type (interaction
   log-OR = +0.26, p < 1e-6), and the benefit appears most concentrated
   in the favorable NPM1+/FLT3-ITD- subgroup (interaction p = 1.4e-4 in
   the unadjusted analysis, p = 8e-7 in the adjusted model).
8. Treatment-biomarker interaction: 7+3 is associated with WORSE response
   in patients with complex karyotype (interaction p = 0.0016).
9. Venetoclax/azacitidine has a small overall positive main effect on
   response (~+0.7 percentage points, p = 0.031) and a positive interaction
   with unfit-for-intensive status (p = 0.036).

Refuted or absent findings of note:
- Targeted FLT3 inhibitors (midostaurin, gilteritinib): NO detectable
  positive interaction with FLT3-ITD or FLT3-TKD in this cohort. The
  treatments appear to be assigned without regard to FLT3 status (~15% of
  FLT3-ITD+ patients receive each, similar to the FLT3-ITD- population).
- Ivosidenib: NO positive interaction with IDH1 mutation. The interaction
  was actually negative (p = 0.027) — IDH1+ patients on ivosidenib responded
  worse than IDH1+ patients off ivosidenib in this cohort. Likely
  reflects a bias in how ivosidenib was assigned (or simply the absence
  of an encoded benefit) and is unlikely to reflect a true clinical
  effect, but it stands as the data reported.
- Enasidenib x IDH2 mutation: NO benefit detected.
- Most comorbidity and demographic main effects (age, sex, secondary AML,
  unfit status, prior therapy lines, smoking, education, rural residence,
  insurance type): NO significant association with objective response
  univariably.
- Most pharmacogenomic SNPs: no detectable signal (1-2 borderline hits
  consistent with multiple-testing chance).
- Negative-control non-AML biomarkers (CA-125, CEA, PSA, HER2, BRAF V600E):
  no association, as expected.
- A small but significant difference in response by race/ethnicity (white
  patients 17.3% vs other groups 14.7-16.7%) was detected in the omnibus
  test (p = 0.010). This effect is small and may be confounded; it should
  be interpreted with caution.

Summary clinical interpretation
-------------------------------
For an AML cohort, the response-prediction signal in this dataset is
dominated by classical prognostic factors (performance status, marrow burden,
nutrition/inflammation markers, WBC) and one treatment-biomarker pair:
venetoclax/azacitidine + NPM1 mutation. IDH1 mutation is itself a
favorable response marker independent of therapy. The targeted-therapy
beneficiary signals expected from real-world AML practice (FLT3 inhibitors
in FLT3-ITD; IDH1 inhibitor in IDH1+; IDH2 inhibitor in IDH2+) are NOT
present in this dataset. The data thus support an interpretation in which
treatment assignment is essentially random with respect to mutation status
and the only encoded predictive interactions are venetoclax/azacitidine x
NPM1 and 7+3 x complex karyotype (negative).
