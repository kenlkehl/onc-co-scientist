Analysis Summary — ds001_aml (50,000 patients; outcome: objective_response, base rate 16.9%)

Across 25 iterations and 113 statistical tests we probed main effects, subgroup
heterogeneity, and treatment-by-biomarker interactions for objective_response
in this AML cohort.

============================================================
ITERATION 1 — Demographics and performance status
============================================================
Hypotheses tested: age_years, sex_female, ecog_ps as main effects on response.

- ECOG performance status was strongly negatively associated with response
  (logistic beta = -0.384, p = 1.8e-97). Each one-unit higher ECOG cuts the
  log-odds of response by ~0.38; this is the single strongest patient-level
  signal in the dataset. SUPPORTED.
- Age (beta = +0.0006, p > 0.1) and female sex (rate diff +0.005, p > 0.1)
  showed no detectable unadjusted association. NOT SUPPORTED.

============================================================
ITERATION 2 — AML driver mutations
============================================================
- idh1_mutation: response 22.1% vs 16.5% (rate diff +0.056, p = 2.9e-18) —
  a strong favorable prognostic signal independent of any treatment.
  SUPPORTED.
- npm1_mutation, flt3_itd, idh2_mutation, tp53_mutation, complex_karyotype:
  no significant unadjusted main effect on response (all p > 0.1).
  NOT SUPPORTED for these.

============================================================
ITERATION 3 — Treatment main effects
============================================================
Tested all six therapies as unadjusted exposure variables.
- treatment_venetoclax_azacitidine showed a small but significant positive
  effect (17.3% vs 16.6%, diff = 0.0074, p = 0.031). Modestly SUPPORTED.
- treatment_midostaurin, treatment_gilteritinib, treatment_ivosidenib,
  treatment_enasidenib, treatment_7plus3: no significant unadjusted main
  effect (all p > 0.1). NOT SUPPORTED at the marginal level — but unadjusted
  comparisons are confounded by indication (each agent is given to specific
  biomarker subgroups).

============================================================
ITERATION 4 — Targeted-therapy x biomarker interactions
============================================================
We tested the standard clinical pairings via logistic interaction terms:
- midostaurin x flt3_itd: interaction beta = -0.067, p = 0.62. NOT SUPPORTED.
- gilteritinib x flt3_itd: interaction beta = +0.094, p = 0.50. NOT SUPPORTED.
- ivosidenib x idh1_mutation: interaction beta = -0.364, p = 0.027. SIGNIFICANT,
  but in the OPPOSITE direction from what is clinically expected: ivosidenib
  is associated with WORSE response in IDH1-mutant patients than would be
  predicted from main effects.
- enasidenib x idh2_mutation: interaction beta = -0.158, p = 0.34.
  NOT SUPPORTED.

============================================================
ITERATION 5 — Lab markers
============================================================
- wbc_k_per_ul (per-unit beta = -0.0022, p = 1.9e-5): higher WBC -> lower
  response. SUPPORTED.
- blast_pct_marrow (beta = -0.0057, p = 1.2e-19): higher marrow blasts ->
  lower response. STRONGLY SUPPORTED.
- albumin_g_dl (beta = +0.115, p = 1.4e-6): higher albumin -> higher
  response. SUPPORTED.
- ldh_u_l, hemoglobin_g_dl, platelets_k_ul: no significant association.

============================================================
ITERATION 6 — Targeted therapy effects within mutation strata
============================================================
- midostaurin within flt3_itd=1: rate diff -0.013, p = 0.27. NOT SUPPORTED.
- gilteritinib within flt3_itd=1: rate diff -0.003, p = 0.81. NOT SUPPORTED.
- ivosidenib within idh1_mutation=1: response rate 17.4% on ivo vs 22.5%
  off ivo (rate diff -0.051, p = 0.042). SIGNIFICANT BUT NEGATIVE: among
  IDH1-mutant patients, those receiving ivosidenib actually responded LESS
  than IDH1-mutant patients not receiving it. This contradicts the clinical
  prior.
- enasidenib within idh2_mutation=1: rate diff -0.022, p = 0.40.
  NOT SUPPORTED.

============================================================
ITERATION 7 — Fitness x intensity interactions
============================================================
- venetoclax/aza x unfit_for_intensive: interaction beta = +0.104, p = 0.036.
  Ven/aza's response benefit is concentrated in unfit patients. SUPPORTED.
- ven/aza within unfit subset: 17.9% vs 16.3%, p = 0.003. SUPPORTED.
- 7+3 x unfit_for_intensive interaction: not significant (p = 0.97).
- 7+3 within fit subset: rate diff +0.004, p = 0.62. NOT SUPPORTED.

============================================================
ITERATION 8 — TP53/complex karyotype prognosis
============================================================
- TP53+complex-karyotype combo: no detectable difference in response vs
  others (rate 16.6% vs 16.9%, p = 0.65). NOT SUPPORTED — this is a
  surprising null given the extreme adverse prognosis of this combination
  clinically.
- 7+3 x tp53 interaction p = 0.95; ven/aza x tp53 interaction p = 0.50.
  NOT SUPPORTED.

============================================================
ITERATION 9 — NPM1 favorable subset
============================================================
- NPM1+/FLT3-ITD- composite: rate diff +0.005, p = 0.43. NOT SUPPORTED.
- 7+3 x NPM1 interaction: p = 0.31. NOT SUPPORTED.

============================================================
ITERATION 10 — Sociodemographic disparities
============================================================
- race_ethnicity: omnibus chi-square p = 0.043 (white 17.3%, hispanic 16.7%,
  asian 16.3%, black 15.7%, other 14.7%). Statistically significant
  heterogeneity but with small absolute spread. WEAKLY SUPPORTED.
- insurance_type, rural_residence, education_years: no significant
  association. NOT SUPPORTED.

============================================================
ITERATION 11 — Comorbidities and prior therapy
============================================================
- diabetes_mellitus, chronic_kidney_disease, heart_failure, prior_malignancy,
  prior_chemotherapy, and prior_lines_of_therapy all showed no significant
  unadjusted association with response (all p > 0.1). NOT SUPPORTED.

============================================================
ITERATION 12 — Inflammation / cachexia
============================================================
- crp_mg_l: beta = -0.0070, p = 2.8e-6 — higher CRP -> lower response.
  SUPPORTED.
- weight_loss_pct_6mo: beta = -0.040, p = 7.4e-35 — strongest cachexia
  signal in the dataset. STRONGLY SUPPORTED.
- nlr, bmi: not significant.

============================================================
ITERATION 13 — Symptom burden
============================================================
- appetite_loss_grade: beta = -0.023, p = 0.038. WEAKLY SUPPORTED.
- fatigue_grade, pain_nrs, dyspnea_grade: not significant.

============================================================
ITERATION 14 — Pharmacogenomic SNP screen (24 SNPs)
============================================================
Two SNPs reached nominal significance (uncorrected) for main effect on
response:
- snp_rs1050828: beta = +0.119, p = 0.005 (positive association).
- snp_rs2032582: beta = -0.048, p = 0.049 (weak negative association).
At Bonferroni-adjusted alpha (0.05/24 = 0.002), only rs1050828 approaches
threshold. The other 22 SNPs were null. WEAKLY SUPPORTED for rs1050828.

============================================================
ITERATION 15 — Top SNP x treatment interactions
============================================================
Tested rs1050828 (top SNP) interactions with treatment_venetoclax_azacitidine,
treatment_7plus3, and treatment_midostaurin. All three interaction p-values
were > 0.5. NOT SUPPORTED.

============================================================
ITERATION 16 — Multivariable model with key covariates
============================================================
Model: response ~ age + ECOG + blasts + albumin + LDH + NPM1 + FLT3-ITD +
TP53 + complex karyotype.
- albumin_g_dl: adjusted beta = +0.117, p = 1.1e-6 (independent prognostic).
  SUPPORTED.
- npm1_mutation, tp53_mutation, age: not independently significant in this
  model. NOT SUPPORTED.

============================================================
ITERATION 17 — Secondary AML
============================================================
- secondary_aml main effect: not significant (p = 0.92). NOT SUPPORTED.
- secondary_aml x ven/aza interaction: p = 0.84. NOT SUPPORTED.

============================================================
ITERATION 18 — Hepatic and renal labs
============================================================
- creatinine_mg_dl, total_bilirubin_mg_dl, ast_u_l, alt_u_l: none reached
  significance. NOT SUPPORTED.

============================================================
ITERATION 19 — Age and ECOG x intensity interactions
============================================================
- age x ven/aza, age x 7+3, ECOG x 7+3: none significant (all p > 0.1).
  NOT SUPPORTED — there is no detectable age- or fitness-graded interaction
  for the intensive vs lower-intensity therapy choice in this dataset.

============================================================
ITERATION 20 — Cell counts
============================================================
- anc_k_ul, alc_k_ul main effects, and ven/aza x platelets interaction: all
  not significant. NOT SUPPORTED.

============================================================
ITERATION 21 — Disease history
============================================================
- years_since_diagnosis, smoking_pack_years: not significant. NOT SUPPORTED.

============================================================
ITERATION 22 — IDH1 in unfit subset; adjusted IDH1 effect
============================================================
- ivosidenib in idh1=1 & unfit=1: rate diff -0.027, p = 0.42. NOT SUPPORTED.
- enasidenib in idh2=1 & unfit=1: rate diff +0.011, p = 0.74. NOT SUPPORTED.
- After adjusting for ivosidenib + age + ECOG + albumin + TP53 + complex
  karyotype + NPM1 + FLT3-ITD, idh1_mutation remains independently
  associated: beta = +0.361, p = 1.2e-17. STRONGLY SUPPORTED. So IDH1
  mutation is itself favorable — independent of whether it is targeted.

ITERATION 23 — Treatment combinations and other interactions
============================================================
- 7+3 + ven/aza combo vs single-agent: rate diff +0.005, p = 0.65.
  NOT SUPPORTED.
- midostaurin x npm1 interaction: p = 0.45. NOT SUPPORTED.

============================================================
ITERATION 24 — Symptom and comorbidity x treatment interactions
============================================================
- pain x 7+3, diabetes x ven/aza, heart_failure x 7+3: all not significant
  (all p > 0.1). NOT SUPPORTED.

============================================================
ITERATION 25 — Comprehensive multivariable model
============================================================
Model: response ~ age + sex + ECOG + albumin + LDH + blasts + WBC + Hgb +
plt + NPM1 + FLT3-ITD + IDH1 + IDH2 + TP53 + complex karyotype + secondary
AML + unfit + all six treatments.
Pseudo-R2 = 0.040.
Independently significant predictors (p < 0.05):
- ecog_ps (negative; p ~ 1e-95)
- albumin_g_dl (positive; beta = +0.118, p = 8.6e-7)
- blast_pct_marrow (negative; p ~ 1e-19)
- wbc_k_per_ul (negative)
- idh1_mutation (positive; beta = +0.360, p = 2.1e-17)
- treatment_venetoclax_azacitidine (small positive)
All other treatments and other mutations did NOT reach independent
significance.

============================================================
OVERALL CONCLUSIONS
============================================================
1. The dominant prognostic axis in this cohort is not which targeted
   therapy is given but baseline patient health: ECOG performance status,
   marrow blast burden, albumin, WBC, CRP, and weight loss are all robust
   predictors of objective response, with effect sizes much larger than any
   single treatment.

2. IDH1 mutation is the strongest molecular favorable factor (~5–6
   percentage-point absolute increase in response, beta ~ +0.36 adjusted),
   and this is INDEPENDENT of receiving ivosidenib. Strikingly, the
   treatment_ivosidenib x idh1_mutation interaction is significantly NEGATIVE
   (p = 0.027), and within IDH1-mutant patients, those given ivosidenib have
   LOWER response (17.4%) than those not given it (22.5%). This contradicts
   the clinical expectation that ivosidenib should preferentially benefit
   IDH1-mutant patients and likely reflects either confounding by indication
   (sicker IDH1+ patients selected for ivosidenib) or a quirk of this
   simulated dataset.

3. Venetoclax/azacitidine x unfit-for-intensive interaction was supported
   (p = 0.036): ven/aza's small benefit is concentrated in the unfit
   population, consistent with its real-world indication.

4. Midostaurin/gilteritinib x FLT3-ITD interactions were NOT detectable —
   the canonical FLT3-inhibitor benefit in FLT3-mutant AML did not emerge in
   this dataset.

5. TP53/complex karyotype, despite their well-known dismal prognosis
   clinically, were not associated with worse response in either marginal or
   adjusted analyses — a notable null in this dataset.

6. Of 24 pharmacogenomic SNPs screened, only snp_rs1050828 reached
   nominal significance for a positive association with response
   (beta = +0.119, p = 0.005), and no SNP x treatment interactions reached
   significance.

7. Sociodemographic disparities were modest: a statistically significant
   omnibus chi-square for race_ethnicity (p = 0.043) but with absolute
   response-rate spread of only ~2.5 percentage points and no signal for
   insurance, rurality, or education.

The strongest, most robust signals were biological/physiological (ECOG,
albumin, blast %, weight loss, CRP, IDH1 mutation), with treatment-specific
effects mostly absent or small. This pattern suggests the dataset is driven
primarily by general fitness/disease-burden prognostics with limited
treatment-specific predictive value baked in.
